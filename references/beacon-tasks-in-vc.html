Benutzerdefinierte Felder erstellen

Mit Video Cloud können Sie benutzerdefinierte Felder verwenden, um Ihren Videos Metadaten hinzuzufügen. Sobald Sie über Videos verfügen, spielen benutzerdefinierte Felder eine große Rolle bei der Vorbereitung dieser Videos für die Verwendung in Brightcove Beacon. Die benutzerdefinierten Felder, die Sie Video Cloud-Videos hinzufügen, die mit Brightcove Beacon funktionieren, fallen in drei Kategorien. Diese Kategorien und eine kurze Erläuterung jeder sind:

  • Allgemein - Informationen über den Videoinhalt, einschließlich:
    • Genre
    • Kurze und lange Beschreibung
    • Produktionsjahr
    • Werfen
    • Schriftsteller
    • usw.
  • Rechteverwaltung - Unterstützt Monetarisierung und Zugriffsbeschränkungen. Zum Beispiel die Art der Werbung und das Datum der Videoverfügbarkeit.
  • Episodendaten - Wenn das Video eine Episode einer Serie ist, definieren Sie die Serie, Staffel und Episodennummer.

Die Erstellen benutzerdefinierter Video Cloud-Felder document dient ausschließlich dazu, Ihnen beim Erstellen der erforderlichen benutzerdefinierten Felder zu helfen.

Diesen benutzerdefinierten Feldern werden in Video Cloud Studio Werte zugewiesen.

Informationen zur Rechteverwaltung hinzufügen

Wie bereits erwähnt, werden benutzerdefinierte Felder verwendet, um Informationen zur Rechteverwaltung hinzuzufügen, die die Monetarisierung und Zugriffsbeschränkungen steuern. Zum Beispiel die Felder

  • Beacon.Rechte.Index.Startdatum
  • Beacon.Rechte.Index.Endtermin

definieren, wann auf ein bestimmtes Video-Asset zugegriffen werden kann.

Video-Playlists erstellen

Playlists werden in Brightcove Beacon bei der Entscheidung, welche Videoinhalte auf einer App-Seite angezeigt werden sollen, stark genutzt. Sie erstellen Wiedergabelisten in Video Cloud Studio.

Im Folgenden finden Sie Ressourcen zum Übertragen Ihrer Videos in Video Cloud:

  • Video Cloud-Grundlagen: Erstellen einer manuellen Wiedergabeliste
  • Video Cloud-Grundlagen: Erstellen einer intelligenten Wiedergabeliste
  • Erstellen und Verwalten von Wiedergabelisten
In Video Cloud Studio erstellte Wiedergabelisten dürfen NUR Videos enthalten. Wenn Sie Wiedergabelisten erstellen möchten, die Serien, Live-Ereignisse usw. enthalten, müssen diese in Brightcove Beacon erstellt werden.

Brightcove Live-Ereignisse erstellen

Erstellen Sie eine SSAI-Anzeigenkonfiguration

In Brightcove Beacon kann serverseitige Anzeigeneinfügung (SSAI) verwendet werden. Um dies zu implementieren, müssen Sie in Video Cloud Studio eine Anzeigenkonfiguration erstellen. Die Implementierung von VOD SSAI Dokument enthält, wie Sie diese Anzeigenkonfiguration erstellen, sowie den End-to-End-Prozess zum Einrichten von VOD SSAI für Brightcove Beacon.
