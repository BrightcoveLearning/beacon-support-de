Video Cloud-Bilder

Wenn ein Video in Video Cloud aufgenommen wird, werden automatisch zwei Bilder erstellt und in den Videoeigenschaften angezeigt. Die beiden erstellten Bilder sind a Poster und ein Miniaturansicht. Die Bilder werden im angezeigt BILDER Abschnitt im Studio. Sie werden diese Upload-Bilder für die Verwendung in Brightcove Beacon verwenden, müssen jedoch für eine ordnungsgemäße Funktion das Standardaufnahmeprofil ändern. Die Schritte im folgenden Warnfeld zeigen Ihnen, wie Sie die Änderung vornehmen. Wenn Sie die Änderung nicht vornehmen, wird das hochgeladene Hochformatbild weiterhin für eine Miniaturansicht angepasst und sieht in Ihrer Brightcove Beacon-App sehr schlecht aus.

Sie MÜSSEN Ihr Standardaufnahmeprofil ändern. Folge diesen Schritten:
  1. Gehe im Studio zu ADMIN → Profile aufnehmen.
  2. Klicken Sie auf den Titel Ihres aktuellen Standardprofils, der mit einem gekennzeichnet ist Upload-Standard Etikett.
  3. Drücke den Duplikat Taste.
  4. Suchen Sie, wo die Bilder sind festgelegt. Die genaue JSON kann je nach dupliziertem Profil etwas anders aussehen als unten gezeigt.
  5. Ändern Sie die Breite und Höhe sowohl für das Poster als auch für die Miniaturbilder wie gezeigt:.

    Alte Einstellungen

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 1280,
  "height": 720,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
  "media_type": "image",
  "format": "jpg",
  "label": "thumbnail",
  "width": 160,
  "height": 90,
  "id": "559a8b0ee4b07cbb40be7d23"
}

    Neue Einstellungen

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 4000,
  "height": 2250,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
    "media_type": "image",
    "format": "jpg",
    "label": "thumbnail",
    "width": 3000,
    "height": 2000,
    "id": "559a8b0ee4b07cbb40be7d23"
}
  6. Klicken Speichern.
  7. Dann klicke auf Machen Sie dies zu meinem Standardprofil Verknüpfung.
  8. Stellen Sie sicher, dass das Profil jetzt als Standard für das Hochladen gekennzeichnet ist.
Sie können jetzt Bilder für Brightcove Beacon mit Video Cloud Studio hochladen.

Verwenden Sie die Studio-Benutzeroberfläche, um zwei Bilder hochzuladen. Sie entscheiden sich für ein Bild, das Ihr Video repräsentiert und machen die entsprechenden Screenshots, die den oben genannten Anforderungen entsprechen. Anschließend laden Sie die beiden Bilder in Video Cloud hoch.

Die Poster und ein Miniaturansicht sind Platzhalter für die in Brightcove Beacon verwendeten Bilder. Befolgen Sie NICHT die Empfehlungen für die Bilder, wie sie in Video Cloud Studio gezeigt werden! Verwenden Sie stattdessen diese Richtlinien:

Poster Miniaturansicht
Querformat (16:9):
Minimum Breite: 1000px Höhe: 563px
Maximal Breite: 4000px Höhe: 2250px
Posterbild (2:3)
Minimum Breite: 1000px Höhe: 1500px
Maximal Breite: 2000px Höhe: 3000px

Das Bild links im Screenshot ist ein Beispiel für die Verwendung des Porträtbilds. Wenn Sie dort ein sehr pixeliges Bild sehen, wissen Sie, dass das Standardaufnahmeprofil aktualisiert werden muss.

Firmenlogo

Sie müssen Ihr Firmenlogo für die Verwendung in Brightcove Beacon Experience angeben. Befolgen Sie diese Richtlinien für das Bild:

Spezifikation Wert
Format .png
Seitenverhältnis 1:1
Größe 1500px x 1500px

Arbeiten in Brightcove Beacon

Bei der Eingabe von Informationen in Brightcove Beacon werden Sie aufgefordert, Nicht-textuelle Daten. Dies bedeutet Bilder für die Assets, die Sie gerade erstellen. Diese Bilder lassen sich in drei Gruppen unterteilen, nämlich:

  • Serien/Jahreszeiten
  • Live-Events
  • Kanallogos

Details für Bilder der Gruppen folgen.

Serien/Jahreszeiten

Für Serien/Saisons erscheint eine Seite ähnlich der folgenden (ohne das rote OPTIONAL Text über die Miniaturansicht , für die Sie kein Bild angeben müssen):

Wenn Sie auf diesen Bildschirm stoßen, geben Sie a JPEG Bild für die Porträt-Poster , Landschaftsposter und Breites Banner den unten stehenden Anforderungen entsprechen. Sie müssen KEIN Bild angeben für Miniaturansicht. Wenn nicht angegeben, wird das Thumbnail-Bild aus dem Portrait-Poster generiert.

Bild Anforderungen
Porträt-Poster Posterbild (2:3)
Minimum Breite: 1000px Höhe: 1500px
Maximal Breite: 2000px Höhe: 3000px
Landschaftsposter Querformat (16:9):
Minimum Breite: 1000px Höhe: 563px
Maximal Breite: 4000px Höhe: 2250px
Breites Banner Querformat (21:9):
Minimum Breite: 1000px Höhe: 428px
Maximal Breite: 4000px Höhe: 1714 Pixel

Live-Events

Bei Live-Events erscheint eine Seite ähnlich der folgenden (ohne das rote OPTIONAL Text über die Miniaturansicht und Breites Banner , für die Sie keine Bilder bereitstellen müssen):

Wenn Sie auf diesen Bildschirm stoßen, geben Sie a JPEG Bild für die Porträt-Poster und Landschaftsposter den unten stehenden Anforderungen entsprechen. Sie müssen KEINE Bilder liefern für Miniaturansicht und Breites Banner. Wenn nicht angegeben, wird das Thumbnail-Bild aus dem Portrait-Poster und das Wide-Banner aus dem Landscape-Poster generiert.

Bild Anforderungen
Porträt-Poster Posterbild (2:3)
Minimum Breite: 1000px Höhe: 1500px
Maximal Breite: 2000px Höhe: 3000px
Landschaftsposter Querformat (16:9):
Minimum Breite: 1000px Höhe: 563px
Maximal Breite: 4000px Höhe: 2250px
