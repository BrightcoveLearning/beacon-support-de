Bekannte Probleme

In diesem Thema werden bekannte Probleme für die {{ site.product }} Anwendung.

Generierte Apps

Streambeschränkungen und schnelles Umschalten zwischen Videos

  1. Wenn ein Zuschauer schnell zwischen Videos wechselt, kann es sein, dass der hier gezeigte Fehler angezeigt wird:

    Dies liegt daran, dass es einige Sekunden dauert, bis der Stream freigegeben wird. Wenn der Zuschauer versucht, ein anderes Video anzusehen, bevor der Stream veröffentlicht wird, tritt eine Streambeschränkung auf und der Fehler wird angezeigt.

Kindersicherung

  1. Alle Inhalte werden Endbenutzern angezeigt (in Listen, Karussells usw.), aber wenn Sie versuchen, das Video abzuspielen, wird die Kindersicherungs-PIN angefordert.

Mobiles Internet

  1. Mobiles Web hat eingeschränkte Unterstützung. Auf Geräten mit einer Breite unter 375 px wird eine Meldung angezeigt, um die entsprechende native Brightcove Beacon-App zu installieren.
