Allgemeine Konfigurationseinstellungen

In diesem Thema erfahren Sie mehr über die allgemeinen Konfigurationseinstellungen, die beim Erstellen oder Ändern Ihrer App in Brightcove Beacon verwendet werden.

Überblick

Dieses Thema ist eine Zusammenstellung der allgemeinen Konfigurationseinstellungen für Ihre Brightcove Beacon-App. Sie finden diese Einstellungen in anderer Dokumentation zum Ausführen bestimmter Aufgaben in Brightcove Beacon, einschließlich der folgenden Registerkartenabschnitte:

  • Layout
  • Seiten
  • Kanäle
  • Filme
  • Wiedergabelisten
  • Programme / Serien
  • Veranstaltungen
  • Handel / Paket / Streams / Erweitert

Rechte & Planung

Die Rechte & Planung Im Abschnitt können Sie Wiedergabeberechtigungen definieren und Veröffentlichungszeitpläne für Ihre App festlegen.

Hier ist ein Beispiel für das Hinzufügen oder Bearbeiten von Seiten:

Diese Tabelle enthält die Felder, die Sie in der Rechte & Planung Tab:

Gebiet Erforderlich Beschreibung
Status Jawohl Wählen Sie einen der folgenden Typen:
  • Veröffentlicht
  • Unveröffentlicht
Rechtetyp Jawohl Geben Sie anhand der folgenden Werte an, ob Ihr Kanal Werbung unterstützt:
  • Kostenlos
  • Werbung - AVOD (Video-On-Demand-Werbung)
  • Werbung - SVOD (Abonnement Video-On-Demand)
  • Erben - Dies bedeutet, dass die Rechte von der Serie vererbt werden, zu der diese Staffel gehört.
Startdatum der Rechte Jawohl Legen Sie das Startdatum und die Startzeit für Ihre App fest. (Ebenfalls, Startzeit)
Enddatum der Rechte Jawohl Legen Sie das Enddatum und die Endzeit für Ihre App fest. (Ebenfalls, Endzeit)
Zu veröffentlichende Geräte Jawohl Fügen Sie die Geräte hinzu, auf denen Sie veröffentlichen möchten. Wähle aus Alle Geräte auswählen / abwählen um eine Liste der verfügbaren Geräte anzuzeigen.
Zulässige Standorte Jawohl

Wählen Sie die geografischen Standorte aus, an denen Sie Benutzern das Anzeigen Ihrer Inhalte ermöglichen möchten. Klicken Sie in das Zulässige Standorte , um eine Liste der Standorte anzuzeigen. Wenn keine Einschränkungen eingegeben werden Welt.

Wenn Sie nicht mindestens einen zulässigen Standort angeben, wird möglicherweise diese Meldung in Ihrer App angezeigt:
Abgelehnte Standorte Nein Wählen Sie die geografischen Standorte aus, an denen Sie NICHT zulassen möchten, dass Benutzer Ihre Inhalte anzeigen. Klicken Sie in das Abgelehnte Standorte , um eine Liste der Standorte anzuzeigen.
Max. Geräte Nein Legen Sie die maximale Anzahl aktiver verbundener Geräte für ein Konto fest. Integer Wert.
Max. Streams Nein Legen Sie die maximale Anzahl von Streams fest, die von demselben Konto aus angesehen werden können. Integer Wert.

Einstellungsbeispiel Max. Streams zu 4 und Max. Geräte zu 6: Das bedeutet, dass 4 Personen Videos streamen (streamen) können, aber Sie können 6 Personen verbunden haben (Geräte).

Textdaten

Die Textdaten Im Abschnitt können Sie anzeigbaren Text hinzufügen. Dies ist der Name, der in Ihrer App angezeigt wird.

Hier ist ein Beispiel für das Hinzufügen oder Bearbeiten von Seiten:

Diese Tabelle enthält die Felder, die Sie in der Textdaten Tab:

Gebiet Erforderlich Anmerkungen
Titel Jawohl Der angezeigte Titel für Ihre App. Für jede von Ihnen definierte Sprache finden Sie ein Eingabefeld.

In der Regel werden Sie am . gefragt Grundinformationen Tab für a Name. Dieser Name ist für interne Referenzen, während die Textdaten Registerkarten Titel wird den Benutzern in der App angezeigt.

Nicht-textuelle Daten

Die Nicht-textuelle Daten Im Abschnitt können Sie Hintergrund- und Symbolbilder für Ihre App hochladen.

Hier ist ein Beispiel für das Hinzufügen oder Bearbeiten von Seiten:

Diese Tabelle enthält die Felder, die Sie in der Nicht-textuelle Daten Tab. Beachten Sie, dass Sie für jede von Ihnen definierte Sprache Bilder hochladen können.

Gebiet Erforderlich Anmerkungen
Symbol Jawohl Das Bild, das Ihre Seite darstellen soll.
Hover-Symbol Jawohl Dieses Bild wird angezeigt, wenn der Benutzer mit der Maus über das Symbolbild fährt.
Horizontaler Hintergrund Jawohl Das Hintergrundbild, das angezeigt wird, wenn sich das Benutzergerät im Querformat befindet.
Vertikaler Hintergrund Jawohl Das Hintergrundbild, das angezeigt wird, wenn sich das Benutzergerät im Hochformat befindet.

Die maximal zulässige Upload-Größe beträgt 500 KB.
Weitere Informationen zum Hochladen von Bildern in Brightcove Beacon finden Sie im Bilder und ihre Spezifikationen dokumentieren.
