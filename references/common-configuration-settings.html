    Allgemeine Konfigurationseinstellungen

    In diesem Thema erfahren Sie mehr über die allgemeinen Konfigurationseinstellungen, die beim Erstellen oder Ändern Ihrer App in Brightcove Beacon verwendet werden.

    Übersicht

    Dieses Thema ist eine Zusammenstellung der allgemeinen Konfigurationseinstellungen, die mit Ihrer Brightcove Beacon-App verknüpft sind. Sie finden diese Einstellungen in einer anderen Dokumentation zum Abschließen bestimmter Aufgaben in Brightcove Beacon, einschließlich der folgenden Registerabschnitte:

    • Layout
    • Facebook Pages
    • Kanäle
    • Filme
    • Wiedergabelisten
    • Programme/ Serien
    • Veranstaltungen
    • Commerce/Paket/Streams/Erweitert

    Rechte & Terminierung

    Im Abschnitt „ Rechte und Planung “ können Sie Wiedergabeberechtigungen definieren und Veröffentlichungspläne für Ihre App festlegen.

    Hier ist ein Beispiel beim Hinzufügen oder Bearbeiten von Pages:

    Diese Tabelle enthält die Felder, die Sie auf der Registerkarte „ Rechte und Planung “ finden:

    Feld Erforderlich Beschreibung
    Status Ja Wählen Sie einen der folgenden Typen aus:
    • Publiziert
    • Unveröffentlicht
    Art der Rechte Ja Geben Sie an, ob Ihr Kanal Werbung aus den folgenden Werten unterstützt:
    • Kostenlos
    • Werbung - AVOD (Werbevideo auf Abruf)
    • Werbung - SVOD (Abonnement-Video-On-Demand)
    • Vererben - Dies bedeutet, dass die Rechte von der Serie übernommen werden, zu der diese Staffel gehört.
    Rechte Startdatum Ja Legen Sie das Startdatum und die Startzeit für Ihre App fest. (Auch Startzeit)
    Enddatum der Rechte Ja Legen Sie das Enddatum und die Endzeit für Ihre App fest. (Auch Endzeit)
    Zu veröffentlichenden Geräte Ja Fügen Sie die Geräte hinzu, auf denen Sie veröffentlichen möchten. Wählen Sie „ Alle Geräte auswählen/Auswahl aufheben“ , um eine Liste der verfügbaren Geräte anzuzeigen.
    Zugelassene Ja

    Wählen Sie die geografischen Standorte aus, an denen Sie Benutzern die Anzeige Ihrer Inhalte ermöglichen möchten. Klicken Sie in das Feld Zulässige Orte , um eine Liste der Standorte anzuzeigen. Wenn keine Einschränkungen sind, betreten Sie World.

    Wenn Sie nicht mindestens einen zulässigen Standort angeben, wird möglicherweise diese Meldung in Ihrer App angezeigt:
    Verweigerte Standorte Nein Wählen Sie die geografischen Standorte aus, an denen Sie Benutzern NICHT erlauben möchten, Ihre Inhalte anzuzeigen. Klicken Sie in das Feld Verweigerte Orte , um eine Liste der Standorte anzuzeigen.
    Max-Geräte Nein Legen Sie die maximale Anzahl aktiver verbundener Geräte für ein Konto fest. Ganzzahliger Wert.
    Max Streams Nein Legen Sie die maximale Anzahl von Streams fest, die vom selben Konto aus überwacht werden können. Ganzzahliger Wert.

    Beispiel für das Setzen von Max Streams auf 4 und Max Devices auf 6: Dies bedeutet, dass 4 Personen Videos streamen können (Streams), Sie können jedoch 6 Personen anschließen (Geräte).

    Textuelle Daten

    Im Abschnitt „ Textdaten “ können Sie anzeigbaren Text hinzufügen. Dies ist der Name, der in Ihrer App angezeigt wird.

    Hier ist ein Beispiel beim Hinzufügen oder Bearbeiten von Pages:

    Diese Tabelle enthält die Felder, die Sie auf der Registerkarte Textdaten finden:

    Feld Erforderlich Hinweise
    Titel Ja Der angezeigte Titel für Ihre App. Sie finden ein Eingabefeld für jede von Ihnen definierte Sprache.

    In der Regel werden Sie auf der Registerkarte „ Grunddaten “ nach einem Namengefragt. Dieser Name ist für interale Verweise gedacht, während der Titel der Registerkarte „ Textdaten “ den Benutzern in der App angezeigt wird.

    Nicht-textuelle Daten

    Im Abschnitt „ Nicht-textuelle Daten “ können Sie Hintergrund- und Symbolbilder für Ihre App hochladen.

    Hier ist ein Beispiel beim Hinzufügen oder Bearbeiten von Pages:

    Diese Tabelle enthält die Felder, die Sie auf der Registerkarte Nicht-Textdaten finden. Beachten Sie, dass Sie Bilder für jede von Ihnen definierte Sprache hochladen können.

    Feld Erforderlich Hinweise
    Symbol Ja Das Bild, das Sie für Ihre Seite darstellen möchten.
    Hover-Symbol Ja Dieses Bild wird angezeigt, wenn der Benutzer mit der Maus über das Symbolbild steht.
    Horizontaler Ja Das Hintergrundbild, das angezeigt wird, wenn sich das Benutzergerät im Querformat befindet.
    Vertikaler Hintergrund Ja Das Hintergrundbild, das angezeigt wird, wenn sich das Benutzergerät im Hochformat befindet.

    Die maximal zulässige Upload-Größe beträgt 500 KB.
    Einzelheiten zum Hochladen von Bildern innerhalb von Brightcove Beacon finden Sie im Dokument Bilder und ihre Spezifikationen .
