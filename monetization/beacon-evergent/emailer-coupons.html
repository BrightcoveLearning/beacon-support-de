Einführung

Das Backoffice-System von Evergent bietet Services, die Sie bei der Verwaltung Ihres Unternehmens unterstützen. In diesem Thema werden wir zwei dieser Dienste behandeln:

  • Der automatisierte E-Mailer
  • Der Gutscheinmanager

E-Mail-Versender

Mit dem E-Mailer können Sie Vorlagen und Zeitpläne für verschiedene vom System generierte E-Mails erstellen. In den folgenden Abschnitten werden die E-Mailer-Operationen detailliert beschrieben.

Nachrichtentypen

Evergent unterstützt die folgenden Nachrichtentypen:

Terminplanung
Wenn die Batchjobs im Systemadministrator geplant werden, wird eine E-Mail an einen Administrator gesendet.
Erinnerung an den Ablauf der Kreditkarte
Wenn auf Geschäftsebene das Kontrollkästchen "Kreditkartenablauferinnerung" aktiviert ist, wird die Erinnerungs-E-Mail an die Kunden gesendet, wenn ihre Kreditkarte abläuft (die E-Mail wird normalerweise einen Monat vor Ablauf der Karte gesendet). In diesem Fall verwendet das System den Nachrichtentyp der E-Mail-Vorlage "Kreditkartenablauferinnerung".
Massen-E-Mails
Wenn ein Unternehmen eine Werbe-E-Mail planen und einen zukünftigen Zeitpunkt für die Ausführung angeben möchte, wird der Nachrichtentyp Massen-E-Mail verwendet. Diese Arten von Massen-E-Mails können an einen Abonnenten einer Geschäftseinheit, einen Abonnenten eines bestimmten Marktgebiets oder einen bestimmten Dienst gesendet werden.
Kreditkartentransaktion
Wenn die Kreditkartenzahlungstransaktion von einem Abonnenten durchgeführt wird, werden die Transaktionsdetails mit dieser E-Mail-Vorlage per E-Mail an den Kunden gesendet.
Ereignisnachricht
E-Mail-Vorlagen vom Typ Ereignisnachricht werden verwendet, um eine E-Mail-Vorlage zu definieren und sie der Ereigniskategorie zuzuordnen, wodurch die Möglichkeit besteht, E-Mail-Benachrichtigungen an Abonnenten auszulösen, sobald das Ereignis auf Kontoebene aufgezeichnet wurde.
Ad hoc
Der Adhoc-Nachrichtentyp soll die einzelnen Nachrichten senden, die nicht mit einer der Ereigniskategoriezuordnungen verknüpft sind.

E-Mail-Versender

Die Registerkarte E-Mail-Versender wird verwendet, um die E-Mail-Vorlagen hinzuzufügen/zu bearbeiten und E-Mails für die Geschäftseinheiten zu planen.

Hinzufügen eines neuen E-Mail-Versenders

Gehen Sie wie folgt vor, um einen neuen E-Mail-Versender hinzuzufügen:

  1. Auf der E-Mail-Versender Klicken Sie auf dem Bildschirm einer Geschäftseinheit auf Hinzufügen () Symbol.
  2. In dem Grundlegende Details geben Sie die erforderlichen Details wie folgt ein:
    1. Auf der Titel Geben Sie im Feld den Titel für die E-Mail ein.
    2. Auf der E-Mail Betreff Geben Sie im Feld den Betreff der E-Mail ein.
    3. Wählen Sie den Nachrichtentyp aus, für den Sie eine E-Mail-Nachricht definieren möchten Nachrichtentyp Dropdown-Liste.
    4. In dem Administrator-E-Mail Geben Sie im Feld die E-Mail-ID ein, von der die E-Mail gesendet werden soll.
    5. In dem Administratorname Geben Sie im Feld den Administratornamen ein.
    6. Wähle aus Ereigniskategorie aus der bereitgestellten Dropdown-Liste.
    7. Geben Sie im die E-Mail-ID der Personen ein, an die die E-Mail gesendet werden soll CC und BCC Felder.
      Notiz: Sie können in den Feldern CC und BCC mehrere E-Mail-IDs angeben, indem Sie die einzelnen E-Mail-IDs durch ein Komma trennen.
  3. Auswählen Klartext oder HTML um die E-Mail im ausgewählten Format zu senden.
  4. In dem E-Mail/Newsletter-Nachricht Geben Sie im Abschnitt E-Mail-Text die Nachrichtenvorlage ein, in der die Nachricht in der E-Mail des Kunden angezeigt werden soll.
  5. Klicken Vorschau neben dem Textfeldbereich, um die E-Mail-Nachricht im Nur-Text- oder HTML-Format anzuzeigen.
  6. Drücke den Schaffen Taste.

E-Mail-Adresse aktualisieren

  1. Navigieren Sie zum E-Mail-Versender Bildschirm einer Geschäftseinheit.
  2. In dem Liste der E-Mail-Vorlagen Klicken Sie im Abschnitt auf die E-Mail-Vorlage, die Sie bearbeiten möchten.
  3. Die E-Mail-Adresse aktualisieren angezeigt wird, nehmen Sie die erforderlichen Änderungen in den verfügbaren Feldern vor und klicken Sie auf Aktualisieren.
Beachten Sie, dass Sie die Felder Nachrichtentyp und Ereigniskategorie auf dem Bildschirm E-Mailer aktualisieren nicht ändern können.

E-Mails planen

Sie können die E-Mails so planen, dass sie später gesendet werden. Dieser Bildschirm dient dazu, die Massen-E-Mails auf einer bestimmten Zeitbasis oder täglich für den DMA oder Dienst zu planen.

E-Mail planen

  1. Auf der E-Mail-Versender Bildschirm, auf dem E-Mail-Vorlagen Klicken Sie auf der Registerkarte auf den Nachrichtentyp E-Mail als Massen-E-Mails.
  2. Drücke den E-Mails planen Tab. Die Liste der geplanten E-Mails wird angezeigt.
  3. Um eine E-Mail zu planen, klicken Sie auf Hinzufügen () Symbol, um das Dialogfeld E-Mail planen zu öffnen.
  4. Geben Sie die E-Mail-ID in das ein Mail senden an Gebiet.
  5. Drücke den Ausführungszeit Feld, um das Datum und die Uhrzeit des Zeitplans auszuwählen.
  6. Um E-Mails täglich für das ausgewählte Datum und die ausgewählte Uhrzeit zu planen, wählen Sie die Tagesablauf Kontrollkästchen.

Löschen einer geplanten E-Mail

  1. Drücke den E-Mail-Vorlagen Registerkarte unter dem E-Mail-Versender Registerkarte aus der linken Menüleiste.
  2. Klicken Sie auf eine Massen-E-Mail (Nachrichtentyp), die Sie entfernen möchten.
  3. Die E-Mail-Adresse aktualisieren Bildschirm erscheint.
  4. Drücke den E-Mails planen Tab.
  5. Klicken Sie auf Löschen () Symbol im Aktionen Spalte, um die geplante E-Mail aus dem Liste der geplanten E-Mails.

Gutscheingenerierung

Die Berechtigung zum Generieren von Coupons basiert auf der BU-Gruppenberechtigung des Benutzers. Angemeldete Benutzer mit dieser Berechtigung können die Coupons generieren und bearbeiten.

Hinzufügen eines neuen Coupons/Gutscheins

  1. Drücke den Gutscheine aus dem linken Menü.
  2. Auf der Gutscheine Klicken Sie auf dem Bildschirm einer Geschäftseinheit auf das Hinzufügen () Symbol. Die Gutscheine Standardbildschirm erscheint.
  3. Klicken Hinzufügen () Symbol zum Generieren des Coupons.
  4. Der Bildschirm zum Generieren neuer Coupons/Gutscheine wird wie unten gezeigt angezeigt.
  5. Geben Sie die Informationen in die Felder wie folgt ein:
    1. Gutschein-Präfix: Präfix des Coupons (z. B.: TestC, Hal, CD, Freitag usw.)
    2. Chargenname: Name des Stapels für die Coupons (Bsp.: Testcoupon, Weihnachten, etc.)
    3. Einzelhändler: Coupons können für die Händlerauswahl generiert werden wie (AT&T, Vodafone, Verizon, Idea usw.)
    4. Anfangsdatum: Das Datum, ab dem der Coupon für den Benutzer verfügbar ist
    5. Haltbarkeitsdatum: Benutzer können den Tag definieren, an dem der Gutschein abläuft
    6. Gutschein-Anforderer: Gibt den Namen des Anforderers an, der die Coupons angefordert hat
    7. Gutschrift als Wallet-Aufladung:
      • Um den Coupon für die Aufladung der Wallet zu generieren, klicken Sie auf den Kippschalter, um ihn zu aktivieren und geben Sie den Betrag ein
      • Um den Coupon als Gutschein zu generieren, klicken Sie auf den Kippschalter, um ihn zu deaktivieren
    8. Rabattart: Definieren Sie den Coupon-Typ – z. B. Kostenlos, Pauschal, Pauschal, Prozentsatz, Prozentsatz und Gültigkeitsverlängerung
    9. Betrag/Prozentsatz: Für den ausgewählten Typ wird das Feld Betrag/Prozent angezeigt
    10. Zeitraum & Dauer: Der angewendete Coupon-Zeitraum kann eine Anzahl von Tagen oder eine Anzahl von Monaten sein, wie die jeweilige Auswahl des Zeitraums die Dauer des Coupons impliziert
    11. Präfix als Gutscheincode erstellen: Wenn Sie dieses Kontrollkästchen aktivieren, wird das Präfix als Gutscheincode gespeichert
    12. Anzahl Coupons: Der Benutzer kann die Anzahl der für den Stapel generierten Coupons angeben. Im Fall von "Präfix als Gutscheincode" ist die Anzahl der Gutscheine "1"
    13. Coupon-Priorität: Wird verwendet, um die generische Coupon-Anzeige auf einer Prioritätsauftragsbasis festzulegen, wie sie im Unternehmen generiert und einem Abonnement- und Angebotstyp zugeordnet ist
    14. Gutscheinlänge: Länge des Gutscheincodes, der generiert werden soll
    15. Mehrere Einlösungen: Wenn dieser Kippschalter ausgewählt ist, kann der Benutzer die maximalen Einlösungen für einen Coupon festlegen
    16. Maximale Einlösungen: Ermöglicht die Anzahl der Einlösungen für einen Coupon
    17. Mehrere Einlösungen durch einen einzelnen Benutzer : Das Kontrollkästchen ermöglicht es Kunden, einen einzelnen Coupon auf ihrem Konto für die maximale Anzahl von Einlösungen durch einen einzelnen Benutzer einzulösen
    18. Mehrere Einlösungen durch einen einzelnen Benutzer : Das Kontrollkästchen ermöglicht es Kunden, einen einzelnen Coupon auf ihrem Konto für die maximale Anzahl von Einlösungen durch einen einzelnen Benutzer einzulösen
    19. Groß-/Kleinschreibung: Aktivieren Sie das Kontrollkästchen, wenn Sie Gutscheincodes in Großbuchstaben generieren möchten
    20. Ist Geschenk: Wenn das Kontrollkästchen aktiviert ist, kann der Coupon an andere weitergegeben werden
    21. Vererbbar: Wenn das Kontrollkästchen aktiviert ist, kann der Coupon vererbbar sein
    22. Sind alle Elemente obligatorisch?: Wenn dieses Kontrollkästchen aktiviert ist, sind alle Elemente obligatorisch
    23. Zahlungsmethode überspringen: Wenn das Kontrollkästchen aktiviert ist, wird die Zahlungsmethode nach dem Einlösen des Coupons übersprungen
    24. Gutscheinbeschreibung: Geben Sie die Gutscheinbeschreibung ein
    25. Kampagneninfo: Geben Sie die Kampagneninformationen ein
    26. Erfolgsmeldung: Geben Sie die Erfolgsmeldung ein, die dem Benutzer angezeigt werden soll.
    27. Warnmeldung: Geben Sie die Warnmeldung ein, die dem Benutzer angezeigt werden soll
    28. Anwendbare Produkte: Der Benutzer kann die Nr. der anwendbaren Produkte auf den Coupon (z. B.: Pauschaler 30% RABATT-Gutschein für das Produkt Neon)
    29. Anwendbare Pakete: Der Benutzer kann die Nr. der Bewerbungspakete zu den Coupons
    30. Angebotsarten: Der Benutzer kann die Angebotstypen für die Coupons auswählen
    31. Anwendbare Zahlungsmodi: Der Benutzer kann die anwendbaren Zahlungsarten für den Coupon auswählen
  6. Drücke den Gutschein erstellen Schaltfläche zum Generieren von Coupons (der Zeitpunkt der Couponerstellung hängt auch von der Anzahl der Coupons ab, die der Benutzer generiert).
  7. Sobald Coupons generiert wurden, Gutscheine erfolgreich generiert Meldung wird angezeigt:

Gutscheinsuche

Durchsuchen Sie die Coupons, indem Sie die verschiedenen Optionen verwenden. Um die Coupons zu durchsuchen, gehen Sie wie folgt vor:

  1. Auf der Gutscheine Geben Sie den Feldwert ein, um nach Chargenname, Präfix, Gutscheincode, Status, Ablaufdatum von/bis zu suchen.
  2. Klicken Sie auf die Schaltfläche Suchen. Die Coupon-Suchergebnisse werden wie unten gezeigt angezeigt.
  3. Klicken Sie auf den Umschalter Enable Advance Search, um die erweiterte Suche zu aktivieren.
  4. Sie können mit dem Coupon-Batch nach den in den Listen aufgeführten anwendbaren Produkten und anwendbaren Zahlungsarten suchen. Um mit Coupon-Batch zu suchen, klicken Sie auf die Schaltfläche Coupon-Batch-Suche.
  5. Die gesuchten Coupons werden in der Suchergebnisliste angezeigt, wie unten gezeigt.
