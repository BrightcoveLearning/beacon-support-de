Schritte zum Erstellen eines neuen Coupons

Gehen Sie folgendermaßen vor, um neue Coupons/Gutscheine in Brightcove Beacon zu erstellen:

  1. Klicken Sie in der Benutzeroberfläche von Brightcove Beacon auf das Handel Tab.
  2. Wählen Sie im linken Menü Gutscheine von Stripe.
  3. Drücke den Neuen Gutschein hinzufügen Taste.
  4. In dem Ausweis Geben Sie im Feld den Code ein, den die Benutzer verwenden werden.
  5. Für die Name verwenden Sie einen benutzerfreundlichen Namen, der in Rechnungen angezeigt werden kann.
  6. Für die Dauer geben Sie die Laufzeit des Coupons in Monaten ein.
  7. In dem Rabattart Sie haben die Möglichkeit, den Rabatt in Währung oder Prozent anzugeben. Wenn der Rabatt beispielsweise 3 Monate kostenlos ist, legen Sie die Dauer auf . fest Wiederholung mit 3 Monaten und Rabattart als 100%.
  8. In dem Einschränkungen Abschnitt können Sie die Anzahl der Einlösungen oder das Datum für die Verwendung des Coupons begrenzen. Nehmen Sie beispielsweise an, dass der Coupon nur bis Ende August verwendet werden kann oder dass nur die ersten 100 Personen den Coupon erhalten.
  9. In dem Pakete/Abonnement verbunden Geben Sie im Abschnitt die Abonnements an, die mit dem Coupon verwendet werden können. Wenn Sie kein Abonnement auswählen, gilt es für alle Paket-Abonnements.
Beachten Sie das Folgende
  • Sobald der Coupon/Gutschein erstellt wurde, kann jeder registrierte Benutzer ihn beim Kauf eines Abonnements einlösen.
