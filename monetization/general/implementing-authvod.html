AuthVOD verstehen

AuthVOD (AUTHenticated Video On Demand) ist ein Monetarisierungspaket, bei dem sich Zuschauer bei der Brightcove Beacon-App authentifizieren müssen, d. h. einen Benutzernamen und ein Kennwort eingeben, um die im Paket enthaltenen Videos anzusehen. Die Videos können je nach Konfiguration dieses Pakets Werbung zeigen oder nicht.

Siehe die Monetarisierungsoptionen verstehen Weitere Informationen zu AuthVOD sowie zu allen Monetarisierungspaketen, die Sie mit Brightcove Beacon verwenden können, finden Sie in diesem Dokument.

Übersicht über die Konfiguration eines AuthVOD-Pakets

Hier sind die allgemeinen Schritte zum Konfigurieren eines AuthVOD-Pakets:

  1. Erstellen Sie in Brightcove Beacon ein unveröffentlicht , kostenloses Paket.
  2. Wenn Anzeigen gewünscht werden, geben Sie beim Erstellen des Pakets einen Anzeigenanbieter an.
  3. Fügen Sie in Video Cloud Studio Daten zu Videos hinzu' benutzerdefinierte Felder, um sie dem AuthVOD-Paket zuzuweisen.
  4. Nehmen Sie in Brightcove Beacon die Videos auf und leeren Sie den Cache.

Erstellen eines Pakets in Brightcove Beacon

Hier sind die detaillierten Schritte, die Sie in Brightcove Beacon ausführen müssen, um ein AuthVOD-Paket zu erstellen:

  1. Drücke den Handel Registerkarte am oberen Rand von Brightcove Beacon.
  2. Auswählen Pakete aus der linken Navigation.
  3. Drücke den Neues Paket hinzufügen Taste.
  4. Auf der Paket tab geben Sie den internen Namen ein und lassen Sie die Status wie Unveröffentlicht.
  5. Wenn Sie Werbung schalten möchten, klicken Sie auf das Jawohl Optionsfeld für Werbung dann wähle eine Anzeigenanbieter aus der Dropdown-Liste.
  6. Auf der Textdaten Geben Sie den für die Betrachter sichtbaren Namen und eine Überschrift über das Paket ein.
  7. Falls gewünscht, liefern Sie ein Bild, das mit den Paketinformationen angezeigt wird.
  8. Auf der Streams Geben Sie auf der Registerkarte Werte für die maximale Anzahl von Geräten ein, die gleichzeitig streamen können, und die maximale Anzahl von Geräten, die eine aktive Verbindung haben können.
  9. Wenn Sie eine genauere Kontrolle der Paketverfügbarkeit wünschen, klicken Sie auf das Jawohl Optionsfeld für Erweiterte Streams? , dann das Formular mit genaueren Verfügbarkeitsoptionen ausfüllen.

Videos zum Paket in Studio hinzufügen

Um dem Paket Inhalt hinzuzufügen, müssen Sie Video Cloud Studio verwenden, um benutzerdefinierten Feldern Werte zuzuweisen. Die beiden benutzerdefinierten Felder, denen Sie für ein AuthVOD-Paket Werte zuweisen müssen, sind:

  • beacon.rights.<counter>.packageName
  • Beacon.rights.<Zähler>.Typ

Die folgenden Schritte beschreiben den Vorgang des Zuweisens von Werten zu diesen benutzerdefinierten Feldern.

Weitere Informationen zur Verwendung der benutzerdefinierten Felder von Video Cloud mit Brightcove Beacon finden Sie unter Erstellen benutzerdefinierter Video Cloud-Felder dokumentieren.

Führen Sie für jedes Video, das Sie in das Paket aufnehmen möchten, die folgenden Schritte aus:

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Klicken Sie im Medienmodul auf den Videonamen, um die Eigenschaften dieses Videos anzuzeigen.
  3. Drücke den BENUTZERDEFINIERTE FELDER' Bearbeiten Taste.
  4. Für die beacon.rights.<counter>.packageName , geben Sie den Namen Ihres zuvor erstellten AuthVOD-Pakets ein.
  5. Für die Beacon.rights.<Zähler>.Typ , Eintreten SVOD.

Videos in Brightcove Beacon aufnehmen

Änderungen, die in Video Cloud Studio an Videos vorgenommen werden, müssen in Brightcove Beacon aufgenommen werden. Dies geschieht automatisch stündlich, Sie können jedoch bei Bedarf auch manuell aufnehmen. Die groben Schritte werden im Folgenden beschrieben.

Ausführliche Informationen zur Aufnahme von Videos finden Sie im Erzwingen einer Synchronisierung dokumentieren.
  1. Klicken Sie in Brightcove Beacon im oberen Menü auf die Registerkarte Tools (Schraubenschlüsselsymbol).
  2. Stellen Sie in der linken Navigation sicher, dass die Einnahme Option ausgewählt ist (es ist die Standardoption).
  3. Drücke den Brightcove-Videos aktualisieren Taste.
  4. Sobald die Aufnahme abgeschlossen ist, können Sie überprüfen, ob Ihre Videos Ihrem AuthVOD-Paket zugewiesen wurden, indem Sie zuerst im oberen Menü auf das Handel Tab.
  5. Stellen Sie in der linken Navigation sicher, dass die Pakete Option ausgewählt ist (es ist die Standardoption).
  6. Klicken Sie auf den Namen Ihres AuthVOD-Pakets.
  7. Klicken Sie auf die Inhalt Registerkarte, um die diesem AuthVOD-Paket zugewiesenen Assets anzuzeigen. Ein Beispiel für diese Registerkarte wird hier gezeigt:
  8. Um das Löschen des Cache zu starten, klicken Sie im oberen Menü auf die Registerkarte Tools (Schraubenschlüsselsymbol).
  9. Klicken Sie in der linken Navigation auf das Zwischenspeicher Möglichkeit.
  10. Drücke den Cache-Bereinigung Taste.
