Überblick

Um eine Anzeigenkonfiguration zu erstellen, müssen Sie Folgendes tun:

  • Fordern Sie von Ihrem Anzeigendienstanbieter eine Anzeigen-Tag-Server-URL an. Diese wird in Video Cloud Studio verwendet, um Ihre SSAI-Anzeigenkonfiguration zu erstellen.
  • Erstellen Sie in Video Cloud Studio Ihre SSAI-Anzeigenkonfiguration. Nach Abschluss der Aufgabe kopieren Sie die Ad-Server-ID, um Ihre Anzeigenkonfiguration in Brightcove Beacon Classic zu erstellen.
  • Erstellen Sie in Brightcove Beacon Classic Ihre Anzeigenkonfiguration, die Sie dann auf Ihre Videoinhalte anwenden können.

Erstellen Sie eine SSAI-Anzeigenkonfigurations-ID

Führen Sie diese Schritte aus, um eine SSAI-Anzeigenkonfiguration zu erstellen:

  1. Navigieren Sie in Video Cloud Studio zu Admin → Serverseitige Anzeigeneinstellungen.
  2. Von dem Anzeigenkonfiguration erstellen Dropdown, wählen Sie VOD
  3. Fügen Sie einen Namen für die Anzeigenkonfiguration hinzu.
  4. Wählen Sie Ihre Anzeigenantwort aus - VMAP oder VAST wird empfohlen. Wenn Sie VAST verwenden, müssen Sie den Videos Cuepoints hinzufügen, um anzugeben, wo Werbeunterbrechungen erscheinen sollen.
  5. Fügen Sie Ihr Anzeigen-Tag ein.
  6. Klicken Speichern.
  7. Suchen Sie Ihre neue Anzeigenkonfiguration in der Tabelle und kopieren Sie Ihre Anzeigenkonfigurations-ID. Sie benötigen diesen Wert für die Beacon-Werbekonfiguration.

Erstellen Sie eine Anzeigenkonfiguration

Gehen Sie folgendermaßen vor, um eine Ankündigungskonfiguration zu erstellen:

  1. Wählen Sie in Brightcove Beacon Classic in der oberen Navigation Werbung .
  2. Klicken Neue Konfiguration hinzufügen.
  3. Benennen Sie die Anzeigenkonfiguration.
  4. Für die Anbieter auswählen Brightcove SSAI
  5. Einfügen in die SSAI-Anzeigenkonfigurations-ID in den Schritten oben kopiert in VOD-Anzeigenkonfigurations-ID.
  6. Klicken Neue Anzeige erstellen.
