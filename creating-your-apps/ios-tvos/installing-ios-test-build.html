Benötigte Information

Um die Brightcove Beacon-App auf Ihrem Apple-Gerät installieren zu können, benötigen Sie:

  • Ein von Brightcove bereitgestellter öffentlicher Link oder eine Einladungs-E-Mail.
  • TestFlight-App auf Ihrem Apple-Gerät installiert.

Von Apple unterstützte Plattformen:

  • iOS-Apps: iPhone, iPad oder iPod touch mit iOS 8 oder höher.
  • tvOS-Apps: Apple TV mit tvOS 12 oder höher.

Eine übergeordnete Ansicht des Prozesses ist:

  • Installieren Sie die TestFlight-App auf Ihrem Gerät.
  • Öffnen Sie den öffentlichen Link oder die Einladungs-E-Mail, die Sie von Brightcove erhalten haben.
  • Lösen Sie den Installationscode für die Brightcove Beacon-App in der TestFlight-Anwendung ein.
  • Führen Sie die App aus.

Installieren der Brightcove Beacon iOS-App

In diesem Abschnitt erfahren Sie, wie Sie die Brightcove Beacon iOS-App herunterladen, um sie auf Ihrem iPhone, iPad oder iPod touch zu testen.

  1. Installieren Testflug auf dem iOS-Gerät, das Sie zum Testen verwenden.
  2. Öffnen Sie Ihre Einladungs-E-Mail oder tippen Sie auf den öffentlichen Link auf Ihrem iOS-Gerät.
  3. Sie erhalten einen Einlösecode in der Einladungs-E-Mail. Kopieren Sie den Code.
  4. Öffne das Testflug App auf Ihrem Apple-Gerät.
  5. Drücke den Tilgen Schaltfläche in der oberen rechten Ecke.
  6. Fügen Sie den Brightcove-Einlösecode ein.
  7. Drücke den Tilgen Taste.

Installieren der Brightcove Beacon tvOS-App

In diesem Abschnitt erfahren Sie, wie Sie die Brightcove Beacon tvOS-App herunterladen, um sie auf Ihrem Apple TV zu testen.

  1. Installieren Testflug auf Ihrem Apple-TV.
  2. Öffnen Sie Ihre Einladungs-E-Mail auf einem Mobilgerät oder Computer.
  3. Sie erhalten einen Einlösecode in der Einladungs-E-Mail. Stellen Sie sicher, dass der Code sichtbar ist, damit Sie ihn später eingeben können.
  4. Öffne das Testflug App auf Ihrem Apple TV.
  5. Klicken Sie in die Tilgen Registerkarte und geben Sie den Brightcove-Einlösungscode ein.
  6. Die Brightcove Beacon-App wird zum Testen auf Ihrem Gerät installiert.
