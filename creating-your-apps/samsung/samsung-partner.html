Überblick

Sie benötigen eine Samsung Apps TV-Partnerschaft und eine Gruppe auf Partnerebene, wenn Sie Ihre App außerhalb der USA starten oder auf bestimmte APIs zugreifen möchten.

Hier ist der Prozess der Partnerschaftsanfrage:

  1. Erstellen Sie eine öffentliche Gruppe, falls Sie noch keine haben. Dies ist für die Partnerschaftsanfrage erforderlich.

  2. Fordern Sie eine Offline-Vereinbarung mit dem Samsung Content Manager an. In einigen Fällen kann der Content Manager auf diese Anforderung verzichten.

  3. Sobald der Samsung Content Manager Ihre Anfrage genehmigt, werden Sie ein Samsung Apps TV-Partner.

Erstellen einer öffentlichen Gruppe

Erstellen Sie zunächst eine öffentliche Gruppe, falls Sie noch keine haben.

  1. Navigieren Sie in einem Browser zum Samsung-Anmeldung Seite.

  2. Melden Sie sich entweder mit Ihrem Konto an oder erstellen Sie ein neues Konto.

  3. Wenn Sie noch kein Mitglied einer öffentlichen Gruppe sind, erstellen Sie eine neue Gruppe. Wählen Sie in der linken Navigation Gruppenmanagement.

  4. Wähle aus Gehe zu "Gruppe erstellen" Taste.

  5. Werte eingeben für Gruppeninformationen und Angaben zum Verkäufer. Wenn Sie fertig sind, wählen Sie die Gruppe erstellen Taste.

Partnerschaft anfragen

Fordern Sie als Nächstes eine Partnerschaft an.

  1. Wählen Sie auf dem Startbildschirm des Entwicklers entweder die Partnerschaft anfordern in der linken Navigation oder die Partnerschaftsanfrage Link aus der oberen Navigation.

  2. Geben Sie die Partnerinformationen ein.

    Die Seite besteht aus drei Eingabebereichen:

    • Informationen zum Samsung Contents Manager
    • Gruppeninformationen
    • Angaben zum Verkäufer

    Je nach Status der Gruppenerstellung sieht die Seite des Anfrageformulars anders aus.

    • Wenn Sie bereits eine Gruppe erstellt haben, werden die Werte sowohl für die Gruppeninformationen als auch für die Verkäuferinformationen von Ihrer Gruppe festgelegt. Hier können Sie Ihre Gruppeninformationen ändern.
    • Wenn Sie keine Gruppe erstellt haben, sind alle Felder leer.

    Für die Registrierung sind sowohl die Felder Gruppeninformationen als auch Verkäuferinformationen erforderlich.

    Da Samsung Partnerschaftsanfragen basierend auf Gruppen verarbeitet, erstellt das Verkäuferbüro Ihre Gruppe als öffentliche Gruppe, bevor Ihre Anfrage geprüft wird.

  3. Wenn du deine nicht kennst Samsung-Inhaltsmanager , senden Sie eine Frage mit 1:1 Fragen und Antworten . Samsung gibt die E-Mail-Adresse Ihres Inhaltsmanagers an.

    • Bewegen Sie den Mauszeiger in der oberen Navigation über Unterstützung.
    • Auswählen 1:1 Fragen und Antworten.
    • Wähle aus Frage stellen Taste.
  4. Wenn Sie mit dem Ausfüllen der Partnerschaftsinformationsseite fertig sind, wählen Sie die Partnerschaft anfordern Taste.

Überprüfung des Partnerschaftsstatus

Schließlich zeigen wir Ihnen, wie Sie Ihren Partnerschaftsstatus überprüfen können.

  1. Sobald der Samsung Content Manager Ihre Anfrage genehmigt, werden Sie ein Samsung Apps TV-Partner.

  2. Sie können bestätigen, dass Sie ein Partnerverkäufer sind, indem Sie die Verkäuferbüro Startseite.
