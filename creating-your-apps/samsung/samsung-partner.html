    Ein Samsung-Partner werden

    In diesem Thema erfahren Sie, wie Sie den Partnerschaftsstatus mit Samsung anfordern können.

    Übersicht

    Sie benötigen eine Samsung Apps TV-Partnerschaft und eine Gruppe auf Partner-Ebene, wenn Sie vorhaben, Ihre App außerhalb der USA zu starten, oder wenn Sie auf bestimmte APIs zugreifen möchten.

    Hier ist der Prozess der Partnerschaftsanfrage:

    1. Wenn Sie noch keine haben, erstellen Sie eine öffentliche Gruppe. Dies ist für die Partnerschaftsanfrage erforderlich.

    2. Fordern Sie eine Offlinevereinbarung mit dem Samsung Content Manager an. In einigen Fällen kann diese Anforderung vom Content Manager aufgehoben werden.

    3. Sobald der Samsung-Content-Manager Ihre Anfrage genehmigt hat, werden Sie ein Samsung Apps TV-Partner.

    Erstellen einer öffentlichen Gruppe

    Erstellen Sie zunächst eine öffentliche Gruppe, wenn Sie noch keine haben.

    1. Navigieren Sie in einem Browser zur Samsung-Sign-Seite .

    2. Melden Sie sich entweder mit Ihrem Konto an oder erstellen Sie ein neues Konto.

    3. Wenn Sie noch kein Mitglied einer öffentlichen Gruppe sind, erstellen Sie eine neue Gruppe. Wählen Sie in der linken Navigation Gruppenverwaltungaus.

    4. Wählen Sie die Schaltfläche Gehe zu „Gruppe erstellen“ .

    5. Geben Sie Werte für Gruppeninformationen und Verkäuferinformationenein. Wenn Sie fertig sind, wählen Sie die Schaltfläche Gruppe erstellen .

    Beantragung einer Partnerschaft

    Beantragen Sie als Nächstes eine Partnerschaft.

    1. Wählen Sie auf dem Startbildschirm des Entwicklers entweder die Schaltfläche Partnerschaft anfordern im linken Navigationsbereich oder den Link Partnerschaftsanfrage in der oberen Navigationsleiste aus.

    2. Geben Sie die Partnerinformationen ein.

      Die Seite besteht aus drei Eingabebereichen:

      • Informationen zum Samsung-Inhaltsmanager
      • Informationen zur Gruppe
      • Informationen für Verkäufer

      Abhängig vom Status Ihrer Gruppenerstellung unterscheidet sich die Seite mit dem Anforderungsformular.

      • Wenn Sie bereits eine Gruppe erstellt haben, werden Werte für Gruppeninformationen und Verkäuferinformationen von Ihrer Gruppe festgelegt. Sie können Ihre Gruppeninformationen hier ändern.
      • Wenn Sie keine Gruppe erstellt haben, sind alle Felder leer.

      Für die Registrierung sind sowohl die Gruppeninformationen als auch die Verkäuferinformationen erforderlich.

      Da Samsung Partnerschaftsanfragen basierend auf einer Gruppe bearbeitet, erstellt das Verkäuferbüro Ihre Gruppe als öffentliche Gruppe, bevor es Ihre Anfrage prüft.

    3. Wenn Sie Ihren Samsung-Inhaltsmanager nicht kennen, senden Sie eine Frage unter Verwendung von 1:1 Q & A. Samsung wird die E-Mail-Adresse Ihres Contents Managers angeben.

      • Bewegen Sie in der oberen Navigation den Mauszeiger über Support.
      • Wählen Sie 1:1 Q & Aaus.
      • Wählen Sie die Schaltfläche „ Frage stellen “.
    4. Wenn Sie mit dem Ausfüllen der Partnerschaftsinformationsseite fertig sind, klicken Sie auf die Schaltfläche Partnerschaft anfordern .

    Verifizieren des Partnerschaftsstatus

    Schließlich zeigen wir Ihnen, wie Sie Ihren Partnerschaftsstatus überprüfen können.

    1. Sobald der Samsung-Content-Manager Ihre Anfrage genehmigt hat, werden Sie ein Samsung Apps TV-Partner.

    2. Sie können überprüfen, ob Sie ein Partner-Verkäufer sind, indem Sie die Startseite des Verkäuferbüros aufrufen.
