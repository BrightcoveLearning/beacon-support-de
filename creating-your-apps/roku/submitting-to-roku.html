    Einreichen von Apps an den Roku-Shop

    In diesem Thema erfahren Sie, wie Sie Ihre Geräte-App beim Roku-Store einreichen.

    Einführung

    Mit Brightcove Beacon erhalten Sie eine generierte Geräte-App, die zum Roku Channel Store hinzugefügt werden kann.

    • Angaben zum Geschäft: https://developer.roku.com/overview
    • Einrichtung des Entwicklerkontos: https://my.roku.com/signup
    • Implementierung der Roku-Suche: https://developer.roku.com/en-gb/docs/developer-program/discovery/search/implementing-search.md
    • Vorgeschlagene Vorlaufzeit für die Genehmigung Drei Wochen
    Beachten Sie, dass es nach der Veröffentlichung bis zu 48 Stunden dauern kann, bis Ihr Kanal auf allen Geräten durchsucht werden kann. Von der Roku-Seite: Mein Kanal wurde gerade veröffentlicht, warum kann ich ihn nicht mit Search finden?

    Erforderliche Informationen für Roku:

    • Anforderungen für Poster und Screenshots
      • Kanal-Poster (540 x 405)
      • Screenshots bis zu 1920 x 1080
    • E-Mail für Entwickler
    • Bevorzugte URL für weitere Informationen
    • Administrativer Kontakt (Name/Telefon/E-Mail)
    • Technischer Kontakt (Name/Telefon/ E-Mail)
    • Client zum Einrichten von App-Logins für QA im Store

    Erste Schritte

    Um auf die Roku-Entwicklerseiten zuzugreifen, führen Sie die folgenden Schritte aus:

    1. Navigieren Sie in einem Browser zur Seite Roku Sign in .

    2. Melden Sie sich entweder mit Ihrem Konto an oder erstellen Sie ein neues Konto.

    Einen Channel hinzufügen

    Erfahre, wie du einen Kanal hinzufügst.

    1. Klicken Sie in der oberen Navigationsleiste auf das Profilsymbol.

    2. Klicken Sie im Dropdown-Menü auf die Option Entwickler .

    3. Wählen Sie in der oberen Navigationsleiste den Link Dashboard aus.

    4. Über das Dashboard können Sie Ihre Kanäle verwalten. Wählen Sie die Schaltfläche „ Kanal hinzufügen “.

    5. Wenn die Option Developer SDK ausgewählt wird, werden Optionen für einen öffentlichen oder nicht zertifizierten Kanal angezeigt.

    6. Wählen Sie Öffentlich aus, wenn der Kanal in den Store gehen soll. Wählen Sie Nicht zertifiziert aus, wenn Sie einen Testkanal erstellen möchten.

      Der Kanalname muss sowohl für öffentliche als auch für nicht zertifizierte Kanäle eindeutig sein.

      Wir empfehlen folgendes:

      • Erstelle deinen Namen des öffentlichen Channels, zum Beispiel Destinations.

      • Erstellen Sie einen nicht zertifizierten Testkanal mit einem Suffix von QA. In diesem Fall würde es also Destinations QAheißen.

      Zunächst erstellen wir den öffentlichen Channel.

    7. Als Nächstes erstellen wir den Testkanal.

    Channel-Eigenschaften

    Eigenschaften sind mit jedem Kanal verknüpft.

    1. Kehren Sie zu Ihrem Entwickler-Dashboard zurück und wählen Sie den Link Meine Kanäle verwalten .

    2. Wähle für deinen Kanal die Schaltfläche „ Vorschau und Update “.

    3. Erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Eigenschaftenaus.

    4. Sie sollten eine Liste der Channel-Eigenschaften sehen. Auf der linken Seite sehen Sie eine Liste der Channel Store-Länder, in denen Ihr Kanal angezeigt wird. Zum Testen können Sie alle Länder auswählen. Wähle für deinen Live-Kanal die Länder aus, in denen dein Kanal angezeigt werden soll.

    5. Wählen Sie für Barrierefreiheit und Compliance Jaaus. Alle Brightcove Beacon-Apps erfüllen diese Anforderungen.

    6. Wählen Sie basierend auf Ihren Inhalten die entsprechende Inhaltsbewertung aus.

    7. Wähle aus, ob ein Kundenkonto für den Zugriff auf deine Channel-Funktionen erforderlich ist.

    8. Geben Sie die URL für die Datenschutzbestimmungen ein, die mit Ihrer gesamten App verknüpft ist.

    9. Der Vanity Access Code wird automatisch von Roku generiert. Sie können diesen Wert belassen.

    10. Wählen Sie die Sprachen aus, die Sie unterstützen möchten.

    11. Wählen Sie für Erforderliche Funktionen zuerst die Klassifizierung aus, die Ihren Kanal am besten beschreibt.

    12. Geben Sie dann an, ob Ihre App eine Internetverbindung benötigt. Wenn Sie mit dem Aktualisieren der Eigenschaften für Ihren Kanal fertig sind, wählen Sie die Schaltfläche Speichern aus.

    Informationen zum Channel Store

    Hier fügst du Beschreibungen und Symbole für deinen Kanal hinzu.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichung zurück, erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Channel-Store-Informationenaus.

    2. Geben Sie für Channel-Store-Informationen die folgenden Feldwerte ein:

      • Kanalname - Dieser sollte bereits mit deinem Kanalnamen gefüllt sein.
      • Beschreibung - Fügen Sie eine detaillierte Beschreibung des Kanals hinzu, die in der Beschreibung des Channel Stores angezeigt wird. (300 Zeichen)
      • Webbeschreibung - Geben Sie eine Kanalbeschreibung für das webbasierte Marketing Ihres Kanals in den USA ein. Dies wird auf Geräten nicht angezeigt. (1500 Zeichen)
      • Channel Poster - Fügen Sie ein Posterbild hinzu, das im Roku Channel Store angezeigt wird. (540 x 405 JPEG oder PNG)
      • Beschreibende Kategorie - Wählen Sie die Kategorie aus, in der der Kanal angezeigt werden soll. (Beispiel: Sport, Drama, Komödie usw.)
      • Inländische Region - Wählen Sie das Land aus, in dem Ihr Kanal als einheimisch und/oder inländisch gilt Ihr Kanal wird in der Kategorie „International“ aller regionalen Channel-Stores angezeigt, die Ihren Kanal enthalten, mit Ausnahme des Channel-Stores, der mit Ihrer Inlandsregion verknüpft ist.

    3. Wenn Sie mit dem Hinzufügen von Channel-Store-Informationen fertig sind, wählen Sie die Schaltfläche Speichern .

    Monetarisierung

    Hier fügen Sie Monetarisierungsinformationen zu Ihrem Kanal hinzu.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichen zurück, erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Monetarisierungaus.

    2. Geben Sie an, ob Ihr Kanal ein Abonnementmodell enthält oder unterstützt.

      Beachten Sie, dass Sie sich selbst dann für das Partner-Auszahlungsprogrammvon Roku anmelden müssen, wenn Sie Brightcove SSAI verwenden. Es ist eine Roku-Store-Voraussetzung, um den Ads-Rückruf während des Anzeigenspiels durchzuführen.

      Für Abonnement-Kanäle geben Sie die folgenden Informationen an:

      • Kaufart - Wie Nutzern der Kanal in Rechnung gestellt wird. (Einmal, monatlich, jährlich usw.)
      • Preisstufe - Wählen Sie einen Wert aus dem Drop-down-Menü aus. Der Kaufpreis für EU Channel Store-Länder beinhaltet die Mehrwertsteuer. Der Erlös basiert auf den Preisen vor Steuern (Netto).

    3. Wenn Sie mit dem Hinzufügen von Monetarisierungsinformationen fertig sind, klicken Sie auf die Schaltfläche Speichern .

    Screenshots

    Hier kannst du Screenshots zu deinem Kanal hinzufügen.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichung zurück, erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Screenshotsaus.

    2. Füge Screenshots hinzu, die im Channel Store erscheinen werden.

      Sie können bis zu 6 Screenshots hinzufügen. Jede muss 1920 x 1080 JPEG- oder PNG-Dateien enthalten.

    3. Wenn Sie mit dem Hinzufügen von Screenshots fertig sind, wählen Sie die Schaltfläche Speichern aus.

    Support-Informationen

    Stellen Sie Kontaktinformationen bereit, um die Unterstützung Ihres Kanals zu unterstützen.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichung zurück, erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Supportinformationenaus.

    2. Geben Sie für Supportinformationen die folgenden Feldwerte ein:

      • Entwickler-E-Mail - Geben Sie Ihre Entwickler-E-Mail-Adresse ein. Wenn dies nicht der Fall ist, aktualisieren Sie Ihre Kontaktinformationen.
      • Bevorzugte URL - Geben Sie die URL für Ihren Kanal an. Dies kann von der Roku-Website oder dem Newsletter verwendet werden, um auf Ihren Kanal zu verlinken.
      • Endbenutzer-Support-Kontakt - Geben Sie eine Webseite, E-Mail-Adresse oder Telefonnummer oder mehrere davon an. Roku wird dies verwenden, wenn der Kundendienst über Ihren Kanal kontaktiert wird. (300 Zeichen) Dies ist erforderlich.

      • Testkonten und Deep-Linking-Parameter - Teilen Sie Brightcove mit, welche Assets Sie als Deep-Links öffnen möchten, und wir werden die Deep-Linking-Parameter aktualisieren. Wenn Sie keine Kontofreigabe einrichten möchten, stellt Brightcove Ihnen die Parameter zur Verfügung.

        Sie geben auch Anmeldeinformationen für 3 Testkonten an. Diese Konten werden für Kundensupport, Demos und QA verwendet. (1500 Zeichen) Wenn Sie Konten mit unterschiedlichen Abonnements haben, müssen Sie diese hier auflisten, damit der Channel Store Zugriff auf alle hat.

      • Adminitstrative Contact - Geben Sie die admininstrativen Kontaktinformationen für Ihre App an.
      • Technischer Kontakt - Geben Sie die technischen Kontaktinformationen für Ihre App an.

    3. Wenn Sie mit dem Hinzufügen von Supportinformationen fertig sind, wählen Sie die Schaltfläche Speichern aus.

    Paket hochladen

    Lade ein Paket auf deinen Kanal hoch.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichen zurück, erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Package Uploadaus.

    2. Zunächst müssen Sie Ihre E-Mail-Adresse bestätigen. Dann können Sie hochladen.

    3. Geben Sie für Package Upload die folgenden Feldwerte ein:

      • Channel-Version - Geben Sie die Version Ihrer App an, beginnend mit Version 1.0.
      • Minimale Firmware - Wählen Sie die Roku-Firmware-Version. Das Paket wird nur auf Geräten ausgeführt, die die Mindest-Firmware-Version oder höher unterstützen. Die neueste stabile Build-Version, die wir empfehlen, ist Roku v9.2 bl.
      • Paketdatei - Die Paketdatei kann von Ihnen oder Brightcove hochgeladen werden. Diese Datei hat einen ähnlichen Namen wie: Beacon_1.0.6138.Pkg.

    4. Wenn Sie mit dem Hinzufügen von Paket-Upload-Informationen fertig sind, klicken Sie auf die Schaltfläche Speichern .

    Statische Analyse

    Erstelle einen Analysebericht über deinen Kanal.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichen zurück, erweitern Sie das Dropdown-Menü Vorschau und Publizieren und wählen Sie Statische Analyseaus.

    2. Wenn Sie ein Paket im vorherigen Abschnitt hochladen, wird Ihr Paket auf Fehler in der Struktur oder Syntax analysiert. Eine Tabelle kann mit Fehlern und Warnungen angezeigt werden.

    Vorschau ansehen und veröffentlichen

    Wenn Sie alle mit Ihrem Kanal verbundenen Formulare ausgefüllt haben, wird Ihnen der Bildschirm Vorschau und Veröffentlichung angezeigt.

    1. Kehren Sie zur Seite Vorschau und Veröffentlichung zurück, erweitern Sie das Dropdown-Menü Vorschau und Veröffentlichen und wählen Sie Vorschau und veröffentlichenaus.

    2. Diese Seite gibt Ihnen eine zusammenfassende Vorschau auf Ihren Kanal.

    3. Wenn du bereit bist, wähle die Schaltfläche „ Senden “, um deinen Channel zu veröffentlichen.

    4. Ihre App wird vom Roku-Store überprüft, um festzustellen, ob sie Probleme hat. Wenn Ihr Kanal aus irgendeinem Grund abgelehnt wird, wird eine E-Mail an die Entwickler-E-Mail gesendet, die Sie auf dem Bildschirm Supportinformationen festgelegt haben. Überprüfen Sie die Gründe für die Ablehnung und erstellen Sie ein Brightcove Supportticket, um die Probleme zu lösen.

    Aktualisiere deinen Kanal

    Wenn Sie eine vorhandene App aktualisieren möchten, führen Sie die folgenden Schritte aus:

    1. Erstellen Sie ein neues Roku-Paket.

      Geben Sie dem neuen Paket eine eindeutige ID und ein eindeutiges Passwort. Stellen Sie Brightcove das neue Paket zusammen mit den Schlüsseln für dieses Paket bereit. Wir benötigen die neuen Informationen, um das neue Paket an den Store zu senden.

      Denken Sie daran, wenn Sie Kanalnamen ändern möchten, müssen diese im Channel Store eindeutig sein.
