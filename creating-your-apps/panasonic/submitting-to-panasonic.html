Einführung

Mit Brightcove Beacon erhalten Sie eine generierte Geräte-App, die zum Panasonic Store hinzugefügt werden kann.

  • Shop-Details: https://developer.vieraconnect.com/
  • Vorgeschlagene Vorlaufzeit für die Genehmigung: Drei Wochen

Erforderliche Informationen für Panasonic:

  • Anwendung
    • Anwendungs-URL und Anwendungs-Staging-URL
  • Beschreibung
    • Maximal 1024 Zeichen
    • Maximal 26 Zeilen
    • Jede Zeile besteht aus maximal 85 Zeichen (oder 44 asiatischen Zeichen)
    • Bitte fügen Sie gegebenenfalls Zeilenumbrüche ein
  • Symbole
    • Es werden vier Symbolbilder benötigt:
      • 160x112 (PNG)
      • Banner-Stil-Symbol 256 x 60 (PNG)
      • 360 x 252 (PNG)
      • Rundes Kreissymbol Durchmesser 336x336 (PNG)
  • Bildschirmfoto
    • Mindestens einer wird benötigt
    • 480w x 270h PNG
    • Sie werden auf der Anwendungsdetailseite im Market angezeigt

Einstieg

Gehen Sie folgendermaßen vor, um auf die Panasonic-Entwicklerseiten zuzugreifen:

  1. Navigieren Sie in einem Browser zum Panasonic Entwickler-Login Seite.

  2. Melden Sie sich entweder mit Ihrem Konto an oder erstellen Sie ein neues Konto.

  3. Wenn Sie sich anmelden, sollten Sie das Bedienfeld sehen.

  4. Lesen Sie den Hinweis rechts neben dem Neue Bewerbung einreichen Taste. Wähle aus Startseite Verknüpfung.

  5. Überprüfen Sie die App-Anforderungen. Keine Sorge, Brightcove erstellt die App mit der erforderlichen Technologie.

Bewerbungsinfo

Die Bewerbungsinfo In diesem Abschnitt geben Sie grundlegende App-Informationen ein.

  1. Wählen Sie in der Systemsteuerung die Neue Bewerbung einreichen Taste.

  2. Sie sollten ein Eingabeformular wie dieses sehen:

  3. In dem Bewerbungsinfo Geben Sie auf der Seite Werte für die erforderlichen Felder ein:

    • Projekttyp - Wählen Sie im Dropdown-Menü die neueste verfügbare HTML-Version aus.
    • Anwendungs-URL - Erstellen Sie zwei Repositorys zum Hosten Ihrer App-Builds von Brightcove: Eine für die Produktion und eine für die Inszenierung. Die Anwendungs-URL ist die Produktions-URL.
    • Staging-URL - Geben Sie die Staging-URL für Ihren App-Build ein.
    • Versionsnummer - Geben Sie die Version Ihrer App ein.

  4. Wähle aus Hinzufügen / Entfernen Taste. Wählen Sie die Sprachen aus, die für Ihre App verfügbar sein sollen. Wähle aus Bestätigen Sie Taste.

  5. Geben Sie Werte für die Felder mit der ausgewählten Sprache ein. In diesem Beispiel, Englisch (Nordamerika) ist ausgewählt.

    • Standardsprache - Wählen Sie aus dem Dropdown-Menü die Ausgangssprache für Ihre App aus. Wenn Ihre App Englisch unterstützt, empfiehlt Panasonic, Englisch ((Nordamerika) oder Englisch (UK-Modell)) zu wählen.
    • Anwendungsname - Geben Sie den Namen ein, den Sie für Ihre App verwenden möchten.
    • Kurzer Name - Geben Sie den Kurznamen ein, den Sie für Ihre App verwenden möchten. Dieser kann mit dem Anwendungsnamen identisch sein.
    • Schlüsselwörter - Geben Sie Schlüsselwörter ein, die von der Market-Suchfunktion verwendet werden sollen, um Ihre App zu finden. Trennen Sie Schlüsselwörter durch ein Komma (z. B. Tischtennis, Sport). Maximal 128 Zeichen.
    • Beschreibung - Geben Sie eine Beschreibung für Ihre App ein. Dies wird auf der Anwendungsdetailseite im Market angezeigt. Klicken Sie in das Eingabefeld für Details.
    • Endbenutzer-Lizenzvereinbarung - Optional. Geben Sie die Endbenutzer-Lizenzvereinbarung ein.

  6. Wählen Sie die nächste Sprache in Ihrer Liste der ausgewählten Sprachen aus und geben Sie die Werte für die im vorherigen Schritt beschriebenen Felder ein.

  7. Wähle aus Nächste Taste.

Zusätzliche Details

Die Zusätzliche Details Im Abschnitt können Sie Support- und Inhaltsdetails hinzufügen.

  1. Nach Auswahl der Nächste Schaltfläche aus dem vorherigen Abschnitt sollten Sie ein Eingabeformular wie dieses sehen:

  2. In dem Zusätzliche Details Geben Sie auf der Seite Werte für die erforderlichen Felder ein:

    • Support-E-Mail - Geben Sie die E-Mail-Adresse an, die Benutzer bei Problemen mit Ihrer App kontaktieren können.
    • Elternbewertung - Erweitern Sie das Dropdown-Menü, um die Definitionen für jede Elternbewertung anzuzeigen. Wählen Sie die für Ihre App geeignete Bewertung aus.
    • Inhaltsbewertung für Brasilien - Dies ist erforderlich, wenn Sie in Brasilien veröffentlichen möchten. Erweitern Sie das Dropdown-Menü, um die Definitionen für jede Brasilien-Bewertung anzuzeigen. Wählen Sie die entsprechende Bewertung aus.
    • Kontakt E-mail - Geben Sie Ihre Kontakt-E-Mail-Adresse ein.
    • Kategorie - Erweitern Sie das Dropdown-Menü, um die Liste der Kategorien anzuzeigen. Wählen Sie die entsprechende Kategorie aus.
    • Preis - Normalerweise können die Apps von Panasonic kostenlos heruntergeladen werden.
    • In-App-Kauf - Wählen Sie aus, ob Ihre App In-App-Käufe haben soll.
    • Marktplatz - Klicken Sie auf die Marktplatz auswählen Schaltfläche, um eine Liste der Länder anzuzeigen. Entweder wählen Alle Länder oder wählen Sie die Länder aus, in denen Ihre App verfügbar sein soll. Wenn Sie fertig sind, wählen Sie die Fertig Taste.
    • Anleitung zur QS - Fügen Sie alle Anweisungen hinzu, die hilfreich sein könnten, einschließlich der Tatsache, dass Sie eine Schaltfläche hinzugefügt, einen Link geändert haben usw. Fügen Sie bei Bedarf Testkonten für Ihre App hinzu.

  3. Wähle aus Nächste Taste.

Assets hochladen

Die Assets hochladen Im Abschnitt können Sie Symbole und Bilder hinzufügen.

  1. Nach Auswahl der Nächste Schaltfläche aus dem vorherigen Abschnitt sollten Sie ein Eingabeformular wie dieses sehen:

  2. In dem Assets hochladen Geben Sie auf der Seite Werte für die erforderlichen Felder ein:

    • App-Beschreibungsdokument - Erstellen Sie ein Dokument, das die Funktionsweise Ihrer App beschreibt. Dies kann dasselbe Dokument sein, das Sie für LG erstellt haben. Wähle aus Durchsuche Schaltfläche, um Ihre Datei zu finden. Wählen Sie dann die Hochladen Taste.

      Beachten Sie, dass nur doc , docx oder pdf Dateien dürfen hochgeladen werden. Die maximale Dateigröße beträgt 8 MB.

    • Symbol 1 - Laden Sie ein Marktsymbol hoch, das auf dem angezeigt wird Auflistungsseite und Anwendungsdetailseite auf dem Markt.

      160w x 112h PNG

    • Symbol 2 (Kreisform) - Laden Sie ein Firefox OS-Symbol hoch, das auf dem angezeigt wird Apps-Dokumentenseite auf dem Markt.

      336 Pixel Durchmesser Kreisform PNG

    • Beispielbild 1 - Dies sind Screenshots Ihrer App, die auf dem Anwendungsdetailseite auf dem Markt. Fügen Sie bis zu 5 Beispielbilder hinzu.

      480w x 270h PNG

  3. Wähle aus Nächste Taste.

AGB zustimmen

Die AGB zustimmen Im Abschnitt können Sie alle Informationen überprüfen, die Sie für Ihre App hinzugefügt haben, bevor Sie sie an den Store senden. Es gibt auch einen Abschnitt mit den Panasonic-Geschäftsbedingungen.

  1. Nach Auswahl der Nächste Schaltfläche aus dem vorherigen Abschnitt sollten Sie ein Eingabeformular wie dieses sehen:

  2. Überprüfen Sie die Informationen und Bilder, die Sie für Ihre App hinzugefügt haben.

  3. Lesen Sie die Allgemeinen Geschäftsbedingungen für Panasonic TV. Aktivieren Sie das Kontrollkästchen, das besagt, dass Sie gelesen und einverstanden sind.

  4. Wähle aus einreichen Taste.

  5. Auf der Anwendungslisten Seite sollten Sie sehen, dass Ihre App als Entwurf gespeichert ist.

  6. Ihre App wird in den QA-Status wechseln, bevor sie live geht.
