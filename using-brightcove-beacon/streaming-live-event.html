    Streaming eines Live-Events

    In diesem Thema erfahren Sie, wie Sie ein Live-Event auf eine Brightcove Beacon-App streamen.

    Übersicht

    Um ein Live-Event auf Ihre Anwendung zu streamen, müssen Sie eine Reihe von Schritten ausführen, um Ihr Video Cloud Live-Konto mit Brightcove Beacon zu verbinden.

    Eine allgemeine Sicht auf den Prozess ist:

    1. Erstelle in Brightcove Beacon ein Event und eine Playlist.
    2. Greifen Sie in Video Cloud auf das Live-Modul zu, um Ihre Veranstaltung zu erstellen und eine Job-ID zu erhalten.
    3. Registrieren Sie in Brightcove Beacon Ihre Live-Veranstaltung mit der Job-ID aus Video Cloud.
    4. Das Live-Video beginnt mit dem Streaming in der Brightcove Beacon App.

    Im letzten Abschnitt dieses Dokuments sehen Sie die Schritte zum Erstellen einer anderen Art von Ereignis in Live mithilfe eines statischen Einstiegspunkts. Dies ermöglicht wiederkehrende Live-Events, die aktiviert und deaktiviert werden können und nur einmal konfiguriert werden müssen.

    Der Rest dieses Dokuments enthält Details für jeden der oben genannten Schritte.

    Erstellen eines Live-Events in Brightcove Beacon

    Befolgen Sie diese Schritte, um ein Event in Brightcove Beacon zu erstellen.

    Nur bei aktivierten Konten mit Brightcove Beacon Live sind die Playlists und Events im Tabs-Bereich verfügbar. Bitte wenden Sie sich an den Brightcove Support, falls Sie sie nicht aktiviert haben.
    1. Klicken Sie in Brightcove Beacon auf die Registerkarte Events , um die Einrichtung eines Events zu starten.
    2. Klicke auf Ein neues Event hinzufügen.
    3. Geben Sie dem Live-Event einen Namen und ein Startdatum für das Ereignis.
    4. Geben Sie auf der Registerkarte Textdaten Ihre Veranstaltungsinformationen ein.
    5. Klicken Sie auf die Registerkarte Nicht-Textdaten .
    6. Geben Sie ein Bild für das Hochformatposter und das Landschaftsposter an, das den folgenden Anforderungen entspricht. Sie müssen KEINE Bilder für Thumbnails und Wide-Bannerbereitstellen.
      Image Anforderungen
      Hochformat-Plak Plakatbild (2:3)
      Mindest Weite: 1000 px Höhe: 563px
      Maximale Weite: 4000px Höhe: 2250 px
      Landschaft-Pl Landschaftsbild (16:9)
      Mindest Weite: 1000 px Höhe: 563px
      Maximale Weite: 4000px Höhe: 2250 px
    7. Klicken Sie auf die Registerkarte „ Rechte und Planung “, wie hier gezeigt:
    8. Fügen Sie Werte für die folgenden Felder hinzu:
      • Rechte-Typ: Wählen Sie die Art der Rechte zwischen Werbung, Abonnement oder Kostenlos.
      • Startzeit: Stellen Sie das Startdatum Ihrer Veranstaltung ein.
      • End-Zeit: Stellen Sie das Enddatum Ihrer Veranstaltung ein.
      • Max. Streams: Legen Sie die maximale Anzahl von Streams fest, die das Live-Event anzeigen dürfen. Die Standardzahl ist 0 (Null).
      • Zu veröffentlichenden Geräte: Fügen Sie die Geräte hinzu, auf denen Sie veröffentlichen möchten. Klicken Sie auf „Alle Geräte auswählen/Auswahl aufheben“ , um eine Liste der verfügbaren Geräte anzuzeigen.
      • Zulässige Standorte: Wählen Sie die Orte aus, an denen Sie Ihre Inhalte streamen möchten, wenn keine Einschränkungen in Worldeintreten.
      • Verweigerte Standorte: Wählen Sie die Orte aus, an denen Sie Ihre Inhalte NICHT streamen möchten.
    9. Klicken Sie auf die Schaltfläche „ Neues Ereignis erstellen“, um Ihre Veranstaltung zu erstellen.

    Hinzufügen des Brightcove Beacon Events zu einer Playlist

    Sobald Sie ein Ereignis erstellt haben, besteht die nächste Aufgabe darin, das Ereignis zu einer Playlist hinzuzufügen. Die folgenden Schritte führen Sie durch diesen Prozess.

    1. Klicke in Brightcove Beacon auf den Tab Playlists .
    2. Klicken Sie auf den Button Neue Playlist hinzufügen .
      Du musst eine Wiedergabeliste in Brightcove Beacon erstellen, die deine Live-Events enthält. VerwendenSie keine Wiedergabeliste aus Video Cloud. Die Verwendung einer Video Cloud-Wiedergabeliste kann Ihre Brightcove Beacon Live-Aufnahmen überschreiben und löschen.
    3. Wählen Sie in Wiedergabelisten aus den Registerkarten für Inhaltstypen die Registerkarte Ereignisse aus.
    4. Suche in der Inhaltsliste nach deinem Event und klicke auf den (+) -Button, um es zur linken Playlist hinzuzufügen. Sie können die Reihenfolge der Ereignisse so positionieren, wie sie in der App angezeigt werden sollen.
    5. Passe die Einstellungen deiner Wiedergabeliste (Zeit, Geräte, Standort usw.) auf der Registerkarte „ Rechte und Planung “ an, wie du es bei deiner Veranstaltung getan hast.
    6. Klicken Sie auf die Schaltfläche Änderungen in Playlist speichern .
    7. Klicken Sie oben rechts auf dem Bildschirm auf die Schaltfläche Extras (Schraubenschlüssel). Leeren Sie dann den Cache.

    Zu diesem Zeitpunkt wird Ihre Veranstaltung als Container in der Brightcove Beacon-App erstellt, wie im linken Bild gezeigt (die Ereignisnamen und Bilder auf dem Bild dienen nur zum Beispiel), klicken Sie auf Ihr Event und Sie sehen etwas Ähnliches wie das Bild auf der rechten Seite. Um tatsächlich mit dem Streamen eines Live-Videos zu beginnen, müssen Sie Ihre Übertragung in der Video Cloud Live Platform einrichten.

    Erstellen einer Live-Veranstaltung in Studio

    Befolgen Sie diese Schritte, um das eigentliche Live-Event zu erstellen, das zu Brightcove Beacon gestreamt wird.

    1. Melden Sie sich bei Video Cloud Studio an und klicken Sie im Navigationsheader auf Live .
    2. Klicken Sie auf der Seite „Ereignisse verwalten “ auf „ Ereignis erstellen“. Die Seite „Live-Ereignis erstellen “ wird geöffnet.
    3. Geben Sie die Veranstaltungsdetails ein.
      • Ereignisname - Name für das Ereignis; dieser Name wird auch im Medienmodul angezeigt, wenn das Live-Ereignis als Video-Asset (max. 255 Zeichen) gespeichert wird
      • Kurzbeschreibung - Beschreibung des Ereignisses (max. 250 Zeichen)
      • Tags - Tags, die mit dem Ereignis verknüpft werden sollen; Tags werden dem Video zugewiesen, wenn das Live-Ereignis als Video-Asset gespeichert wird (maximal 128 Zeichen und 1200 Tags)
      • Referenz-ID - Referenz-ID für das Ereignis; muss eindeutig sein (max. 150 Zeichen)
    4. Klicken Sie auf Erlaubt , wenn Sie das Streamen der Live-Veranstaltung auf Social-Media-Plattformen aktivieren möchten, andernfalls klicken Sie auf Nicht zulässig. Dies kann erfolgen, sobald das Ereignis mit dem Streaming begonnen hat oder mithilfe des Social-Moduls.
    5. Wählen Sie ein Live-Ingest-Profilaus. In der Dropdown-Liste werden alle Live-Aufgaufen-Profile aufgeführt. Das Aufgabe-Profil steuert die Qualität und Anzahl der erstellten Live-Darstellungen. Wenn Sie auf den Link Profile verwalten klicken, wird der Ingest-Profileditor in einem anderen Browser-Tab geöffnet. Beachten Sie, dass Sie Kontoadministrator sein müssen, um auf den Ingest-Profileditor zugreifen zu können. Klicken Sie auf den Link RENDITIONS , um die Formatvarianten anzuzeigen, die mit dem ausgewählten Ingest-Profil erstellt werden.
      Die Bitrate und die Rechnungsdetails der drei Stufen lauten wie folgt:
      • Tier 1 erlaubt bis zu 5.500 Kbit/s und wird mit 1,0 x Stunden aufgeladen
      • Tier 2 erlaubt bis zu 10.000 Kbit/s und wird mit 1,5 x Stunden aufgeladen
      • Tier 3 erlaubt bis zu 20.000 Kbit/s und wird mit 2,0x Stunden aufgeladen
    6. Wählen Sie Event nach Abschluss des Ereignisses in Video-Asset konvertieren , um eine VOD-Version des Live-Events in Ihrem Video Cloud-Konto zu speichern. Wenn diese Option ausgewählt ist, wird die höchste Wiedergabeversion aus dem ausgewählten Live Ingest-Profil als Quelle für das VOD-Video verwendet. Das standardmäßige Auftrittenprofil für das Konto wird beim Erstellen von Formatvarianten verwendet. Beachten Sie, dass die VOD-Version nicht verfügbar ist, sobald das Live-Ereignis endet, das Video nach Abschluss des Aufnahmvorgangs verfügbar ist.
      Hinweis: Diese Option wird bei Verwendung eines wiederkehrenden Ereignisses nicht unterstützt.
    7. Klicken Sie auf ADVANCED OPTIONS und sehen Sie sich Nur einer davon wird Ihre Aufmerksamkeit benötigen.
    8. Klicken Sie in das Kontrollkästchen Benachrichtigungen zum Stream-Status aktivieren und fügen Sie die Benachrichtigungs-URL von Brightcove Beacon CMS hinzu (Ihr Brightcove Support-Mitarbeiter wird diese URL bereitstellen).
    9. Klicken Sie auf Create Event , um das Live-Event zu starten. Die Control Room-Seite wird mit Details zum Live-Ereignis geöffnet, einschließlich des Streaming-Endpunkts (RTMP-URL) und des Stream-Namens , die von Ihrem Encoder verwendet werden. Die Verarbeitung Ihrer Streams dauert ungefähr eine Minute.
      Um mit dem Streaming auf der Live-Plattform zu beginnen, benötigen Sie einen Encoder. Informationen zur Verwendung eines Encoders finden Sie in der folgenden Dokumentation:

      Schritt für Schritt: Verwenden des Live-Moduls mit Telestream Wirecast

      Schritt für Schritt: Verwenden des Live-Moduls mit Open Broadcaster Software (OBS)

    10. Kopieren Sie im Kontrollraum die Job-ID und die Video-ID aus dem Testereignisfenster. Sie benötigen beide, um das Live-Ereignis in den Brightcove Beacon-Plattformeinstellungen zu konfigurieren.
    11. Gehen Sie als Nächstes zurück zu Brightcove Beacon und klicken Sie auf die Registerkarte Event . Wählen Sie das Ereignis aus, das Sie zuvor dort erstellt haben, und fügen Sie in den Einstellungen der Grunddaten die Job-ID Ihres Live-Ereignisses ein.
    12. Wechseln Sie zur Registerkarte Streams und erstellen Sie einen neuen Stream, indem Sie auf die Schaltfläche + Neuen Stream hinzufügen klicken.
    13. Fügen Sie im Feld Stream Video ID die Video-ID aus dem Video Cloud-Ereignis ein, das Sie zuvor erstellt haben. Geben Sie auch die folgenden Informationen über den Stream ein:
      • Wählen Sie die Anzahl der Geräte aus, auf denen Sie Ihr Video veröffentlichen können.
      • Stellen Sie den Streamtyp im Dropdown-Menü auf Live ein.
      • Und klicken Sie auf die Schaltfläche Ereignis aktualisieren , bevor Sie die Seite verlassen.
    14. Klicken Sie oben rechts auf dem Bildschirm auf die Schaltfläche Extras (Schraubenschlüssel). Leeren Sie dann den Cache.
    15. Öffnen Sie Ihre Brightcove Beacon App und wählen Sie Ihre Veranstaltung aus (die Ereignisnamen und Bilder auf dem Bild dienen nur zum Beispiel).
    16. Klicken Sie auf die Schaltfläche „ Abspielen “, und Sie sollten jetzt in der Brightcove Beacon App Ihr Live-Event-Streaming von Video Cloud sehen können.

    Sobald Ihr Event endet, können Sie es ganz oder teilweise als Clip speichern, der zu einem Video-Asset in Video Cloud oder einem abspielbaren Stream wird, der in Brightcove Beacon angezeigt werden kann. Der Prozess wird im nächsten Abschnitt dieses Dokuments beschrieben.

    Erstellen eines Clips aus Ihrem Live-Event

    Sie müssen einen Clip Ihres Brightcove Beacon Live-Events erstellen, wenn Sie das Video zum späteren Streaming als aufgezeichnetes VOD-Video behalten möchten.

    Das Live-Modul bietet die Möglichkeit, Videoclips aus abgeschlossenen Ereignissen zu erstellen. Clips können auch erstellt werden, während ein Live-Event läuft.

    Eine vollständige Liste der Clipping-Funktionen finden Sie in der Dokumentation Erstellen eines Videoclips mit dem Live-Modul .

    1. Wechseln Sie im Live-Modul auf die Registerkarte Clip erstellen im linken Bereich.
    2. Legen Sie mit den Clipping-Tasten oder Tastenkombinationen die Start- und Endpunkte für den Clip fest. Sie können auch einfach auf die Schaltfläche Speichern klicken, um das gesamte Video beizubehalten.
    3. Geben Sie die Eigenschaften des Clip-Videos ein.
      • Clipname - Name des Clips
      • Kurzbeschreibung - Beschreibung des Clips
      • Schlagworte - Tags, die mit dem Clip verknüpft werden sollen
      • Referenz-ID - Referenz-ID zum Zuweisen zum Clip
      • Quellenwerbeblengeninhalt entfernen - Diese Option entfernt Werbeblöcke, die innerhalb des Clips ausgelöst werden, wobei nur Programminhalte beibehalten werden
        Hinweis: Das Kontrollkästchen „Quellenwerbebruch entfernen“ wird nur angezeigt, wenn der Stream SSAI-aktiviert ist.
    4. Klicken Sie auf „ Clip erstellen Der Transcodierungsprozess für den Clip beginnt. Clips werden unter Verwendung des standardmäßigen Aufgabe-Profils für das Konto transcodiert.

    Hinzufügen Ihres Clips zu Brightcove Beacon als spielbares Stream-Video

    Hier verwenden Sie die ID Ihres neu erstellten Clip-Videos und machen es zu einem Stream, der in Brightcove Beacon abgespielt werden kann.

    1. Kopieren Sie die Nummern-ID Ihres erstellten Clips, während Sie sich noch im Live-Modul von Video Cloud Studio befinden.
    2. Wechseln Sie auf der Brightcove Beacon-Plattform zum Tab Ereignisse und wählen Sie das Streams-Menü aus. Sie sehen das folgende Formular:
    3. Fügen Sie die ID , die Sie gerade von der Live-Plattform von Studio kopiert haben, in das Feld Stream Video ID ein.
    4. Wechseln Sie aus dem Dropdown-Menü Streamtyp Live in Stream.
    5. Klicken Sie auf die Registerkarte Grunddaten und ändern Sie den Status von Beendet in Fertig.
    6. Bevor Sie fortfahren, klicken Sie unten auf der Seite auf die Schaltfläche „ Ereignis aktualisieren “. Dadurch wird die Schaltfläche „ Abspielen “ in der Brightcove Beacon-App verfügbar, um die Aufzeichnung des Events anzuzeigen.

    (Statischer Einstiegspunkt) SEP Live-Ereignisse

    SEP Live Events sind wiederkehrende Events, die Sie beispielsweise bei einem wöchentlichen Programm immer wieder aktivieren/deaktivieren können, wenn Ihre Veranstaltung immer wieder beginnt. Dieses Ereignis kann als eine für alle Live Job ID verwendet werden, die in der Brightcove Beacon-App an mehrere Kanäle übertragen werden können.

    • Beachten Sie, dass beim Erstellen eines wiederkehrenden Ereignisses automatisch ein Video im Medienmodul dafür erstellt wird. Löschen Sie dieses Video nicht, da es jedes Mal benötigt wird, wenn das Ereignis erneut stattfindet.
    • Das wiederkehrende Ereignis sollte aktiviert werden, bevor der Encoder gestartet wird.

    Befolgen Sie diese Schritte, um ein SEP (Static Entry Point) Live-Event in Brightcove Beacon zu erstellen.

    1. Erstellen Sie auf der Live-Plattform eine neue Veranstaltung. Weitere Informationen zum Erstellen einer Veranstaltung finden Sie im Abschnitt Erstellen einer neuen Veranstaltung im Dokument Erstellen und Verwalten von Live-Events mit dem Live Module .
    2. Erweitern Sie die ERWEITERTEN OPTIONEN und aktivieren Sie die Option Dies als wiederkehrendes Ereignis mit einem statischen Einstiegspunkt (SEP) erstellen. Weitere Informationen zu SEP finden Sie im Abschnitt Arbeiten mit wiederkehrenden Ereignissen im Dokument Erstellen und Verwalten von Live-Events mit dem Live-Modul .
    3. Wenn Sie bereits ein SEP-Event haben, klicken Sie auf den Reiter Standby , und Sie können es im Live Control Room aktivieren/deaktivieren .
    4. Kopieren Sie Ihre Live-Job-ID.
    5. Kehren Sie zur Brightcove Beacon-Plattform zurück und konfigurieren Sie Ihr Ereignis, indem Sie die Job-ID des wiederkehrenden Ereignisses einfügen. Geben Sie auch einen Namen und ein Startdatum für das Ereignisein.
    6. Das Datum und die Uhrzeit Ihrer Veranstaltung müssen im folgenden Format vorliegen.
    7. Klicken Sie auf die Schaltfläche „ Ereignis aktualisieren “.
    Brightcove Beacon CMS automatisiert die Statusübergänge für Live-Ereignisse, aber in Static Entry Points haben Sie möglicherweise mehrere Brightcove Beacon-Ereignisse mit demselben SEP verknüpft. Wenn es Probleme mit der Automatisierung gibt, die zu spät kommt und sich überschneidet, können Sie den CMS-Übergang anpassen, indem Sie den Statusmanuell ändern. Zum Beispiel von Verfügbar bis Fertig im Feld Status .

    Sie können diesen Prozess für alle gewünschten Live-Events auf der Brightcove Beacon-Plattform durchführen. Alle von ihnen werden von derselben Live-Übertragung gestreamt, aber jede mit dem unterschiedlichen Datum und der anderen Uhrzeit, die Sie in den Ereigniseinstellungen angepasst haben.
