Trailer hinzufügen

Das Hinzufügen eines Trailers für einen Beacon-Film ist einfach. Folgen Sie den unteren Schritten.

  1. Laden Sie das Trailervideo in Ihr Video Cloud-Konto hoch.
  2. Sie machen nicht müssen dem Trailer-Video spezielle benutzerdefinierte Beacon-Felder hinzufügen. Alles was du brauchst ist die Video-ID:
  3. Für das Video, das für den Beacon-Film verwendet wird, fügen Sie das folgende benutzerdefinierte Feld hinzu:
    Benutzerdefiniertes Feld für Trailer
    Anzeigename Interner Name Wert
    beacon.trailer.id beacon_trailer_id (Video-ID für Trailer)
