Wählen Sie die Sprachen für Ihre Apps

Während des Pre-Sales-Prozesses haben Sie Ihre Absicht erklärt, Ihre Apps in mehrere Sprachen zu übersetzen. Zu den Übersetzungsmöglichkeiten gehören die App-Felder und Inhaltsmetadaten.

App-Übersetzungen werden durchgeführt, indem Sie die Übersetzung der App-Felder in Ihrer App-Erfahrung anfordern. Inhaltsübersetzungen sind die Metadaten, die Sie in das CMS eingeben. Es enthält Textdaten wie Titel und Beschreibungen sowie Bilder, die Text enthalten.

Das Definieren der Sprachen, die Sie in Ihren Brightcove Beacon-Apps verwenden möchten, ist Teil des Onboarding-Prozesses. Wenn Sie nach dem Onboarding-Prozess mehrere Sprachen verwenden möchten oder Sprachen hinzufügen möchten, für die Übersetzungen vorliegen, wenden Sie sich bitte an Ihren Brightcove-Supportmitarbeiter. Für neue Sprachübersetzungen wenden Sie sich bitte an den Brightcove-Support.

Was wird übersetzt?

Es gibt zwei Teile der Apps, die übersetzt werden/können, nämlich:

  • Übersetzungen der Feldnamen, Schaltflächen und Beschreibungen in den Apps können von Ihnen bereitgestellt oder von Brightcove für Sie übersetzt werden.
  • Übersetzungen von von Ihnen definierten Elementen, z. B. Seitennamen oder Video-Metadaten, und Bilder werden von Ihnen in Brightcove Beacon eingegeben.

Wie Ihre Zuschauer eine Sprache für die App auswählen

Es gibt drei Möglichkeiten, wie Ihre Zuschauer die Sprache für die App auswählen können. Sie sind:

  • Wählen Sie die Sprache im Einstellungen Seite
  • Stellen Sie die Sprache des Browsers ein (falls zutreffend)
  • Legen Sie die Sprache für Ihr Gerät fest (falls zutreffend)

Ändern Sie die Sprache in der App

Zuschauer können die Sprache in der Brightcove Beacon-App selbst auswählen. Das Folgende zeigt den Prozess, den der Betrachter unter Verwendung der Einstellungen Seite, um die App-Sprache von Englisch auf Deutsch zu ändern:

Stellen Sie die bevorzugte Sprache im Browser ein

Wenn ein Betrachter in einem Browser eine bevorzugte Sprache einstellt und diese Sprache von der Brightcove Beacon-App unterstützt wird, verwendet die App automatisch diese Sprache. Der Betrachter müsste die Sprache in der Einstellungen Standardseite.

Notiz: Wenn in der tatsächlichen Brightcove Beacon-App eine Sprache eingestellt ist, überschreibt dies die Browsereinstellungen.

Alle gängigen Browser haben eine Möglichkeit, die bevorzugte Sprache einzustellen. So wird beispielsweise in Chrome die Einstellung im Fortschrittlich Abschnitt der Einstellungen Möglichkeit:

Stellen Sie die Sprache auf Ihrem Gerät ein

Gehen Sie beispielsweise auf einem iPhone zu Einstellungen → Allgemein → Sprache & Region

Notiz: Wenn in der tatsächlichen Brightcove Beacon-App eine Sprache eingestellt ist, werden die Geräteeinstellungen dadurch überschrieben.

So wählen Ihre Zuschauer eine Video-Asset-Sprache aus

Zusätzlich zu den in den Apps verfügbaren Sprachen stehen Ihnen möglicherweise mehrsprachige Untertitel und Audiospuren für Ihre Video-Assets zur Verfügung. Wenn für das Video mehrere Sprachen verfügbar sind, wählt der Betrachter während der Videowiedergabe die gewünschte Sprache aus. Während der Wiedergabe ist ein Zahnradsymbol verfügbar. Wenn Sie auf dieses Symbol klicken, wird ein Dialogfeld angezeigt und der Zuschauer kann die Sprache für Untertitel und/oder Audiospur auswählen, wie hier gezeigt:

Sprachen für Videometadaten definieren

Der grundlegende Prozess zum Lokalisieren von Video-Metadaten ist:

  • Konfigurieren Sie die gewünschte Sprache in Administrator von Video Cloud Studio Modul.
  • Wählen Sie auf der Seite mit den Videoeigenschaften die Sprache aus, für die Sie lokalisierte Metadaten eingeben möchten.
  • Geben Sie die lokalisierten Metadaten ein.

Weitere Informationen zur Verwendung mehrsprachiger Metadaten finden Sie im Arbeiten mit mehrsprachigen Metadaten dokumentieren.

Der erste Schritt zur Verwendung mehrsprachiger Metadaten besteht darin, die Sprache auszuwählen, in der die Metadaten in Video Cloud Studio angezeigt werden sollen, indem Sie zu . navigieren Admin → Spracheinstellungen. Sie fügen zuerst die Sprache hinzu und speichern sie dann, wie hier gezeigt:
