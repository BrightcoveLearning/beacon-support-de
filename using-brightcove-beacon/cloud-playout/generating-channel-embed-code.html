Generieren des Einbettungscodes für einen Cloud Playout-Kanal

Um den Einbettungscode für einen Cloud Playout-Kanal zu generieren, verwenden Sie das Live-Modul. Ausführliche Informationen zur Verwendung des Live-Moduls finden Sie unter Erstellen und Verwalten von Live-Events mit dem Live-Modul.

Es stehen zwei Arten von Einbettungscodes zur Verfügung:

  • Standard - iframe-Einbettungscode. Für die meisten Fälle empfohlen. Dieser Typ platziert den Player in einem <iframe>-Tag.
  • Fortschrittlich - In-Page-Einbettungscode. Ermöglicht dem Player, direkt in der HTML-Seite zu existieren, nicht in einem Iframe. Dies bietet Entwicklern die Vorteile des einfachen Zugriffs auf den Player und die zugehörigen Eigenschaften und Ereignisse.

Sie haben auch die Möglichkeit, die URL zum veröffentlichten Player abzurufen. Dies ist nützlich, um das Ereignis in einem Browser anzuzeigen.

Gehen Sie folgendermaßen vor, um den Einbettungscode für ein Live-Ereignis zu generieren:

  1. Öffnen Sie das Live-Modul.
  2. Suchen Sie das Live-Ereignis, zu dem der Kanal streamen wird. Normalerweise findet die Veranstaltung im Stehen zu Registerkarte und haben den Namen Kanal es sei denn, Sie haben beim Erstellen des Kanals ein bestimmtes Ereignis zum Streamen ausgewählt.
  3. Klicken Sie auf den Veranstaltungsnamen, um die Kontrollraum Seite.
  4. Klicken Veröffentlichen und einbetten > Player-Konfiguration in der linken Navigation.
    Notiz: Wenn der Kanal so konfiguriert ist, dass serverseitige Anzeigen (SSAI) aktiviert werden, Brightcove Player Version 6.18.3 und höher muss verwendet werden. Nur Spieler der Version 6.18.3 und höher werden in der Dropdown-Liste angezeigt.
  5. Drücke den Spieler-URL um eine Vorschau des Ereignisses in einem Browser anzuzeigen oder den entsprechenden Einbettungscode zu kopieren.

Wenn das Programm erstellt wurde mit Anzeigenschaltung Option auf gesetzt Auf , enthalten die Veröffentlichungsoptionen a Wählen Sie Anzeigenkonfiguration Dropdown-Liste, in der die zu verwendende Anzeigenkonfiguration ausgewählt werden kann.
