Einführung

Je nachdem, wie Sie Ihren Cloud Playout-Kanal konfiguriert haben, können Sie verschiedene Inhalte hinzufügen. VOD-Inhalte und Playlists können zu jedem Kanal hinzugefügt werden. Darüber hinaus können Sie Live-Ereignisse hinzufügen, wenn Sie den Kanal so konfiguriert haben, dass dies unterstützt wird. Die folgenden Abschnitte enthalten Schritte zum Hinzufügen jeder Art von Inhalt.

Beachten Sie, dass Sie auch Stoßfänger hinzufügen zu deinem Kanal und serverseitige Anzeigen , wenn der Kanal dafür konfiguriert wurde. Weitere Informationen dazu finden Sie in diesen Themen.

Hinzufügen von VOD-Inhalten

Nachdem Sie Ihren Kanal erstellt haben oder wenn Sie die Kanaldetails aufrufen, werden auf dem Bildschirm die Kanaleinstellungen angezeigt:

Sie können Details bearbeiten, aber der wichtigste nächste Schritt besteht darin, dem Kanal Inhalte hinzuzufügen.

Schritte zum Hinzufügen von Videos (VOD)

  1. Klicken Videos hinzufügen (Wenn Sie sich auf der Kanallistenseite befinden, klicken Sie auf den Kanalnamen, um die Seite mit den Kanaleinstellungen zu öffnen, und klicken Sie dann auf Videos hinzufügen).
  2. Klicke auf das Kontrollkästchen neben den Videos, die du dem Kanal hinzufügen möchtest.
  3. Klicken Videos hinzufügen)
  4. Klicken Speichern.
  5. Die Programmliste des Senders erscheint:

    Beachten Sie, dass Start- und Endzeiten automatisch aus der Videodauer berechnet werden. Sie können Videos per Drag & Drop in die Liste ziehen, um sie neu anzuordnen, und die Start- und Endzeiten werden automatisch angepasst.

Playlists hinzufügen

Du kannst auch ganze Playlists in deinen Kanal importieren. Beachten Sie die folgenden Bedingungen:

  • Sie können sowohl EXPLICIT (manuell) als auch intelligente Wiedergabelisten importieren.
  • EIN Schnappschuss der Playlist im aktuellen Zustand importiert werden (d.h. zukünftige Änderungen an der Playlist werden nicht in den Kanalinhalten widergespiegelt werden).
  • Es kann jeweils nur eine Playlist importiert werden.
  • Wenn die importierte Playlist ein Live-Ereignis enthält, wird ein identisches neues Live-Ereignis mit gleichem Namen, gleicher Beschreibung und gleicher Dauer erstellt. Dies liegt daran, dass ein Live-Event immer eine 1:1-Beziehung mit dem Kanal hat.
  • Die Videos in der Playlist werden hinzugefügt bis zum Ende des aktuellen Kanalprogramms.
  • Sie können VOD-Inhalte sowie Wiedergabelisten hinzufügen, und Duplikate sind erlaubt.
  • Es gibt eine Validierungsprüfung des Gesamtvermögenslimits (100). Wenn der Import das Limit von 100 Assets überschreitet, schlägt der Import fehl.
  • Notiz: Wenn die Live-Eingabe für den Kanal nicht aktiviert ist, können Sie dennoch eine Wiedergabeliste importieren, die Live-Ereignisse enthält, indem Sie die Live-Ereignisse vom Import ausschließen.
  • Notiz: Wenn es sich um einen SSAI-aktivierten Kanal handelt, sind Live-Events nicht erlaubt. Sie können weiterhin eine Playlist importieren, die Live-Ereignisse enthält, indem Sie die Live-Ereignisse vom Import ausschließen.
  • Notiz: Es sind nur geeignete Inhalte mit einer MP4-Wiedergabe zulässig. Eine Warnung weist darauf hin, welche Assets nicht importiert werden dürfen – nach Bestätigung werden nur die berechtigten Assets importiert.

Playlist-Schritte hinzufügen

  1. Klicken Wiedergabelisten importieren.
  2. Klicken Importieren für die Playlist, die Sie importieren möchten.
  3. Wenn Meldungen angezeigt werden, dass einige Elemente in der Wiedergabeliste nicht importiert werden können, klicken Sie auf Fortfahren wenn Sie die unterstützten Videos importieren möchten.
  4. Klicken Speichern.
  5. Wenn der Import erfolgreich war, sehen Sie ihn in Ihrer Programmliste:

Live-Ereignisse hinzufügen

Hinweis: Dies gilt nur, wenn der Kanal für Live-Ereignisse aktiviert ist.

  1. Klicken
  2. Geben Sie im Dialogfeld Live-Ereignis hinzufügen die Name , Dauer , und Beschreibung für das Live-Event:

    Beachten Sie, dass die Mindestdauer für ein Live-Ereignis 5 Minuten beträgt und die Dauer nur in 5-Minuten-Intervallen erhöht oder verringert werden kann - Bei Bedarf kann die Dauer inline im Textfeld bearbeitet werden, um die Granularität der Zeit anzupassen. Die Dauer eines Live-Events kann jederzeit geändert werden, auch während es gerade läuft.

  3. Nachdem Sie Live-Ereignisse hinzugefügt haben, werden sie zusammen mit VOD-Elementen in der Programmliste angezeigt:
  4. Sie können Live-Ereignisse ziehen, um die Liste wie bei Videos neu anzuordnen. Sie können auch die Schaltfläche, um die Dauer eines Live-Ereignisses zu ändern.

Siehe auch Hinzufügen von Bumpern zu Inhalten zum Hinzufügen von Bumpern zu VOD- oder Live-Event-Elementen.
