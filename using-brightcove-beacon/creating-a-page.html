Erstellen einer neuen Seite

In diesem Thema erfahren Sie, wie Sie eine neue Brightcove Beacon-Seite erstellen.

Überblick

Auf der Brightcove Beacon-Seite wählen Benutzer Ihre Videoinhalte aus und sehen sie sich an. Karussells zeigen eine Gruppe von Videos oder Wiedergabelisten an, die Sie in Video Cloud Studio definiert haben. Mithilfe der Brightcove Beacon-Verwaltungskonsole können Sie eine neue Seite mit einem Inhaltskarussell erstellen.

Seite erstellen

Gehen Sie folgendermaßen vor, um eine neue Seite zu erstellen:

  1. Melden Sie sich bei Brightcove Beacon an.

  2. Wählen Sie in der oberen Navigation Seiten.

  3. Wähle aus Neue Seite hinzufügen Schaltfläche, um eine neue Seite zu erstellen.

  4. Für die Grundinformationen Geben Sie auf der Registerkarte die folgenden Feldwerte ein:

    • Systemtitel - Interner Systemseitenname. Dies wird den Benutzern nicht angezeigt, wird jedoch verwendet, um eine benutzerfreundliche URL zu generieren. Sie geben in einem späteren Schritt einen sichtbaren Titel ein.
    • Befehl - Seitenreihenfolge auf Ihrer Homepage
    • Genre-Typ - Werte:
      • Meister - Hauptseite
      • Kind - Untergeordnete Seite
    • Navigationstyp - Wählen Sie aus einer Liste von Seitenvorlagen aus
    • Präsentationstyp - Werte:
      • Symboltext - Zeigen Sie ein Symbol mit Text darin an
      • Symbol - Zeigen Sie ein Symbol an
      • Text - Text anzeigen
    • Bildschirmgestaltung - Werte: (nicht jeder Navigationstyp hat ein Bildschirmlayout)
      • Bildschirmlayout mit Titel - Layout mit Titeln
      • Bildschirmlayout ohne Titel - Layout ohne Titel
    • Ist es zu Hause? - Wählen Sie aus, ob dies die Startseite ist
  5. Als nächstes fügen wir der Seite einen Block hinzu. Blöcke sind Container für deine Playlists. Einige Navigationstypen haben keine Blöcke, wie z. B. Anmelde- und Abmeldetypen. Wir haben ausgewählt a Navigationstyp mit einem Wert von Bildschirm.

  6. Wähle aus Block hinzufügen Taste. Wenn Sie die Schaltfläche auswählen, werden die Blocktypen auf der rechten Seite angezeigt.

  7. Wählen Sie den gewünschten Blocktyp aus. In diesem Fall wählen wir die Listenblock. Wähle aus Block hinzufügen Schaltfläche, die dem gewünschten Block zugeordnet ist.

  8. Wähle aus Layout hinzufügen Taste. Wenn Sie die Schaltfläche auswählen, werden die Layouttypen auf der rechten Seite angezeigt.

  9. Wählen Sie den gewünschten Layouttyp aus, egal ob Featured, Karussell oder Raster. In diesem Beispiel wählen wir die Karussell-Querformat-Etiketten In Layout.

  10. Wähle aus Playlist hinzufügen Taste. Wenn Sie die Schaltfläche auswählen, werden die Wiedergabelistentypen auf der rechten Seite angezeigt.

  11. Wählen Sie eine Wiedergabeliste. In diesem Beispiel wählen wir die Playlist hinzufügen Schaltfläche, die mit dem verbunden ist Humor Wiedergabeliste.

    Sie erstellen Wiedergabelisten mit Video Cloud Studio, und sie werden hier angezeigt, sobald Sie mit Ihrem Video Cloud-Konto synchronisiert haben.

  12. Wir können sehen, dass die Humor Playlist wurde unserem Karussell hinzugefügt. Sie können die Elemente per Drag & Drop neu anordnen.

  13. Für die Textdaten , Nicht-textuelle Daten , Stichworte und Rechte & Planung Registerkarten, siehe die Allgemeine Konfigurationseinstellungen dokumentieren.

  14. Wähle aus Neue Seite erstellen Schaltfläche, um Ihre neue Seite zu erstellen. Sie sind fertig. Sie sollten das neue Karussell auf Ihrer Brightcove Beacon-Seite sehen.
