Hinzufügen eines Karussells zu einer vorhandenen Seite

In diesem Thema erfahren Sie, wie Sie einer Brightcove Beacon-Seite eine Gruppe von Videos in einem Karussell hinzufügen.

Überblick

Auf der Brightcove Beacon-Seite wählen Benutzer Ihre Videoinhalte aus und sehen sie sich an. Karussells zeigen eine Gruppe von Videos oder Wiedergabelisten an, die Sie in Video Cloud Studio definiert haben. Mithilfe der Brightcove Beacon-Verwaltungskonsole können Sie Karussells zu einer vorhandenen Seite hinzufügen.

Karussell hinzufügen

Gehen Sie folgendermaßen vor, um ein Karussell zu einer vorhandenen Seite hinzuzufügen:

  1. Melden Sie sich bei Brightcove Beacon an.

  2. Wählen Sie in der oberen Navigation Seiten.

  3. Du kannst den ... benutzen Neue Seite hinzufügen Schaltfläche, um eine neue Seite zu erstellen. Nachdem Sie Seiten definiert haben, können Sie ganz einfach ein Karussell zu einer vorhandenen Seite hinzufügen. Wählen Sie die Seite aus, die Sie aktualisieren möchten. Hier wählen wir den Link für die Heim Seite.

  4. Beachten Sie, dass die Grundinformationen ist bereits besiedelt.

  5. Wähle aus Block hinzufügen Taste.

  6. Wählen Sie den gewünschten Blocktyp aus. In diesem Fall wählen wir die Listenblock. Wähle aus Block hinzufügen Schaltfläche, die dem gewünschten Block zugeordnet ist.

  7. Wähle aus Layout hinzufügen Taste.

  8. Wählen Sie den gewünschten Layouttyp aus, egal ob Featured, Karussell oder Raster. In diesem Beispiel wählen wir die Karussell-Querformat-Etiketten In Layout.

  9. Nachdem Sie nun den Container für das Karussell definiert haben, können Sie Inhalte hinzufügen. Wählen Sie aus dem Listenblockelement die Playlist hinzufügen Taste.

  10. Wählen Sie eine Wiedergabeliste. In diesem Beispiel wählen wir die Playlist hinzufügen Schaltfläche, die mit dem verbunden ist Humor Wiedergabeliste.

    Sie erstellen Wiedergabelisten mit Video Cloud Studio. Sie werden hier angezeigt, sobald Sie mit Ihrem Video Cloud-Konto synchronisiert haben.

  11. Wir können sehen, dass die Humor Playlist wurde unserem Karussell hinzugefügt. Sie können die Elemente per Drag & Drop neu anordnen. Wähle aus Seite aktualisieren , um das neue Karussell zur Seite hinzuzufügen.

  12. Sie sind fertig. Sie sollten das neue Karussell auf Ihrer Brightcove Beacon-Seite sehen. Weitere Informationen zu Feldwerten finden Sie in den folgenden Abschnitten.
