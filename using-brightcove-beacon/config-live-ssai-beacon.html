Dieser Live-Anzeigen-Konfigurationsprozess ist nur erforderlich, wenn der Live-Kanal mit SSAI aktiviert wurde. Wenn nicht, sind diese Schritte nicht erforderlich.

Voraussetzungen

  • Erstellen Sie einen Live-Kanal, der mit SSAI in Video Cloud aktiviert ist.
  • Erstellen und richten Sie das Anzeigen-Tag in Video Cloud ein.
  • Erstellen Sie einen Kanal in Brightcove Beacon.
  • Erstellen Sie die Ankündigungskonfiguration in Brightcove Beacon.
  • Legen Sie die Rechte und Planung im Kanal fest.

Konfigurieren der Video Cloud-Anzeige mit Brightcove Beacon

  1. Wählen Sie in Video Cloud Studio die Option LIVE Modul.
  2. Erstellen oder öffnen Sie den Live-SSAI-Kanal, den Sie mit Brightcove Beacon verwenden möchten.
  3. In dem Kontrollraum , klicken Veröffentlichen und einbetten von der linken Tafel.
  4. In dem Live-Player-Optionen Abschnitt, wählen Sie aus der Dropdown-Liste die Anzeigenkonfiguration Sie einstellen möchten.
  5. Einmal eine neue ausgewählt Standard-Einbettungscode wird auftauchen.
  6. Extrahieren Sie die adConfigId indem Sie einen Teil der URL kopieren, die der adConfigId zugewiesen ist, wie im Bild gezeigt.

Erstellen Sie die Anzeige in Brightcove Beacon

  1. Kehren Sie zur Brightcove Beacon-Plattform zurück.
  2. Klicken Werbung.
  3. Erstellen Sie die Anzeigenkonfiguration, die Sie festlegen möchten, oder wählen Sie sie aus.
  4. Geben Sie für eine neue Anzeige ein Name.
  5. Stellen Sie sicher, dass die Anbieter ist eingestellt auf Brightcove SSAI.
  6. In dem Live-Konfigurations-ID Feld, fügen Sie das ein adConfigId URL, die Sie im vorherigen Abschnitt dieses Dokuments kopiert haben.
  7. Klicken Neue Anzeige erstellen.

Festlegen von Rechten und Planung in Brightcove Beacon

  1. Klicken Sie in der oberen Navigation auf Kanäle.
  2. Wählen Sie den Kanal aus, dessen SSAI-Konfiguration Sie einstellen möchten.
  3. Drücke den Rechte & Planung Tab.
  4. In Rechtetyp , auswählen Werbung - AVOD.
  5. In Anzeigenkonfiguration , wählen Sie aus der Dropdown-Liste die gerade erstellte Anzeigenkonfiguration aus.
  6. Klicken Neuen Kanal erstellen um Ihre Arbeit zu speichern.
