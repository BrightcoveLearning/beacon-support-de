Überblick

Playlists sind eine Sammlung von Videos und anderen Entitäten, die auf zwei Arten für Brightcove Beacon erstellt werden:

  1. Aus Ihrer Video Cloud-Bibliothek in Brightcove Beacon importiert. In diesem Fall sind die Elemente in der Playlist NUR Videos.
  2. Erstellt in Brightcove Beacon mit dem Wiedergabelisten Tab. Beim Erstellen der Wiedergabeliste in Brightcove Beacon können Sie die aus Video Cloud importierten Videos (Filme/Episoden) verwenden und außerdem Serien, Staffeln, Kanäle und Live-Ereignisse hinzufügen. Diese Methode steht im Mittelpunkt dieses Dokuments.
Beachten Sie das Folgende:
  • Brightcove empfiehlt, Wiedergabelisten, die nur Videos enthalten, in Video Cloud zu erstellen.
  • Nicht alle Funktionen, die in der Wiedergabelisten Registerkarte ist zu diesem Zeitpunkt verwendbar. In diesem Dokument wird deutlich gemacht, welche Formularelemente Sie ausfüllen sollten.

Abschnitt Grunddaten

Gehen Sie folgendermaßen vor, um eine Wiedergabeliste in Brightcove Beacon zu erstellen:

  1. Melden Sie sich bei Brightcove Beacon an.
  2. Wählen Sie in der oberen Navigation Wiedergabelisten.

  3. Drücke den Neue Playlist hinzufügen Taste.

  4. Von dem Grundinformationen Registerkarte, definieren Sie den Namen und die Anzeigeart der Wiedergabeliste und fügen dann Inhalte hinzu. Beachten Sie, dass die meisten Funktionen in diesem Abschnitt derzeit nicht verwendet werden können. Die folgenden Schritte beziehen sich natürlich nur auf die verwendbaren Elemente.

  5. In dem Name fügen Sie einen internen Systemnamen für Ihre Wiedergabeliste hinzu. Dies ist auch der Name, der für Analysen verwendet wird. Beachten Sie, dass Sie den sichtbaren Titel für die Wiedergabeliste in einem späteren Schritt hinzufügen.
    Es wird empfohlen, die in Brightcove Beacon erstellten Wiedergabelisten im Vergleich zu Video Cloud irgendwie zuzuordnen. Platzieren Sie beispielsweise einfach ein Sternchen (* ) vor dem Namen der Wiedergabeliste, wenn die Wiedergabeliste in Brightcove Beacon erstellt wird. Natürlich muss es kein Sternchen sein, aber wählen Sie eine Notation, damit Sie beim Aktualisieren einer Wiedergabeliste zu Brightcove Beacon wechseln Wiedergabelisten Registerkarte oder Video Cloud Studio, um Änderungen vorzunehmen.
  6. Wählen Sie ein Layout aus Alle Playlist-Layouts anzeigen. Dies ist das Layout, das verwendet wird, wenn Zuschauer auf das Alle ansehen Schaltfläche in den Apps. Wählen Sie als Best Practice ein Raster aus.
  7. Um mit dem Hinzufügen von Inhalten zu Ihrer Playlist zu beginnen, klicken Sie auf das Vermögenswerte Tab. Sie sehen, dass Sie aus sechs Asset-Typen auswählen können.
    Sie befinden sich jetzt an dem Punkt, an dem Sie Playlist-Optionen sehen, nämlich Filme , Serie , Jahreszeiten , Veranstaltungen und Kanäle , die Sie in Video Cloud Studio nicht auswählen können. Auf diese Typen kann in Video Cloud Studio NICHT zugegriffen werden, daher müssen Sie die Wiedergabeliste in Brightcove Beacon erstellen.
  8. Um das gewünschte Asset auszuwählen, klicken Sie zuerst auf den Asset-Typ und dann auf das neben dem eigentlichen Vermögenswert.

Textdaten

Die Textdaten Im Abschnitt können Sie anzeigbaren Text für Ihre Wiedergabeliste hinzufügen.

  1. Wähle aus Textdaten Tab.

  2. Fügen Sie Werte für die folgenden Felder hinzu:

    Gebiet Erforderlich Anmerkungen
    Titel Jawohl Der angezeigte Titel für Ihre Playlist.
    Beschreibung Unzutreffend Dieses Feld wird derzeit nicht verwendet.
    Untertitel Unzutreffend Dieses Feld wird derzeit nicht verwendet.

Nicht-textuelle Daten

Die Nicht-textuelle Daten Registerkarte ist zu diesem Zeitpunkt nicht funktionsfähig.

Stichworte

Die Stichworte Registerkarte ist zu diesem Zeitpunkt nicht funktionsfähig.

Rechte & Planung

Die Rechte & Planung Im Abschnitt legen Sie den Veröffentlichungsstatus und die Verfügbarkeit fest.

  1. Wähle aus Rechte & Planung Tab.

  2. Seien Sie sicher, dass Status ist eingestellt auf Veröffentlicht für Ihre Playlists.
  3. Für die Verfügbarkeitsdetails , verwenden Sie Folgendes als Leitfaden, um die Informationen zu vervollständigen:
    Gebiet Erforderlich Anmerkungen
    Rechtetyp Jawohl Auswählen Kostenlos.
    Startzeit Jawohl Planen Sie eine Startzeit für Ihre Playlist.
    Endzeit Jawohl Planen Sie eine Endzeit für Ihre Playlist.
    Max. Streams N / A Zur Zeit nicht funktionsfähig.
    Zu veröffentlichende Geräte Jawohl Fügen Sie die Geräte hinzu, auf denen Sie veröffentlichen möchten. Wähle aus Alle Geräte auswählen/abwählen um eine Liste der verfügbaren Geräte anzuzeigen.
    Zulässige Standorte Jawohl

    Wählen Sie die Orte aus, an denen Sie Ihre Inhalte streamen möchten, wenn keine Einschränkungen eingegeben werden Welt.

    Wenn Sie nicht mindestens einen zulässigen Standort angeben, wird möglicherweise diese Meldung in Ihrer App angezeigt:
    Abgelehnte Standorte Nein Wählen Sie die Orte aus, an denen Sie Ihre Inhalte NICHT streamen möchten.

    Um geografische Standorte hinzuzufügen oder zu bearbeiten, navigieren Sie zu Einstellungen und dann Standorte .
  4. Klicken Sie unbedingt auf Mache eine neue Abspielliste Schaltfläche, um Ihre Arbeit zu speichern.
