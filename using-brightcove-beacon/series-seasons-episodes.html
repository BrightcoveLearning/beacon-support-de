Überblick

EIN Serie ist eine Sammlung verwandter Videos, die in einer bestimmten Reihenfolge angezeigt werden und normalerweise ein gemeinsames Thema haben. Beim Erstellen einer Serie müssen Sie die Beziehungen zwischen der Serie, ihren Staffeln und den zugehörigen Episoden definieren.

EIN Jahreszeit ist eine Gruppe von Videos, die einen bestimmten Zeitraum abdecken. Jede Serie kann eine oder mehrere Staffeln haben.

Ein Folge ist ein individuelles Video. Sie können sich das als einen bestimmten Film vorstellen.

Nachdem Sie Ihre Serien und Staffeln definiert und Ihre Folgen aufgenommen haben, können sie in Ihrer Brightcove Beacon-App verwendet werden. Gehen Sie dazu folgendermaßen vor:

  1. Serie (Brightcove Beacon)

    Erstellen Sie in Brightcove Beacon die Serie. Hier benennen Sie die Serie und geben Informationen wie Verfügbarkeit und Bilder an.

  2. Jahreszeiten (Brightcove Beacon)

    Erstellen Sie in Brightcove Beacon die Jahreszeiten, die sich auf die Serie beziehen. Sie werden serienähnliche Informationen bereitstellen, zusätzlich zur Auswahl der Serie, zu der die Saison gehört.

  3. Folgen (Video-Cloud)

    Die Wahrheit über den Inhalt der Brightcove Beacon-Episode liegt in Video Cloud.

    Weisen Sie in Video Cloud Studio die Videoepisoden der entsprechenden Staffel zu, indem Sie die in benutzerdefinierten Feldern angegebenen Werte verwenden. Diese benutzerdefinierten Felder umfassen:

    • beacon.episode.number
    • beacon.episode.seasonNumber
    • beacon.episode.serieName

  4. Kehren Sie zu Brightcove Beacon zurück und nehmen Sie Ihre Videoepisoden auf.

Erstellen einer Serie

Gehen Sie folgendermaßen vor, um eine Serie zu erstellen:

  1. Melden Sie sich bei Brightcove Beacon an und wählen Sie Programme.
  2. Wählen Sie in der linken Navigation Serie.
  3. Drücke den Neue Serie hinzufügen Taste.

  4. Auf der Grundinformationen Registerkarte, füllen Sie Folgendes aus:

    Formularfeldname Beschreibung / Anforderungen
    Name Die Kennung der Serie, der kundenseitige Name wird später in der angegeben Textdaten Sektion
    Produktionsjahr Das Jahr, in dem die Serie produziert wurde; diese Informationen erscheinen in der Mehr Info Bildschirm
    Produktionsland Das Land, in dem die Serie produziert wird; diese Informationen erscheinen in der Mehr Info Bildschirm
    Genres Die Genres der Serie; Beachten Sie, dass Sie nach Bedarf Genres erstellen können:
    Untergeordnetes Genre Nicht ausfüllen, diese Daten werden nicht verwendet
    Genre-Sprachvorschau Die Sprache der Vorschau
    Art der Episodensortierung So sortieren Sie Episoden in der App-Benutzeroberfläche
    Werfen Die Metadaten unter WERFEN erscheint in der Mehr Info Bildschirm.
  5. Klicken Sie auf die Textdaten Tab.
  6. Füllen Sie das aus Titel (das ist der dem Betrachter zugewandte Name für die Serie), Beschreibung und Zusammenfassung. Diese Daten werden in der Brightcove Beacon-App angezeigt.
  7. Klicken Sie auf die Nicht-textuelle Daten Tab.
  8. Liefern Sie ein Bild für Folgendes (das den unten aufgeführten Anforderungen entspricht):
    • Porträt-Poster (erforderlich)
    • Landschaftsposter (erforderlich)
    • Miniaturansicht
    • Breites Banner (erforderlich)

    Wenn nicht angegeben, wird das Thumbnail-Bild aus dem Portrait-Poster generiert.

    Maximal Breite: 4000px Höhe: 2250px

  9. Die Informationen zu den Stichworte Registerkarte wird derzeit nicht verwendet, geben Sie daher keine Informationen auf dieser Registerkarte ein.
  10. Die Informationen zu den Metadaten Registerkarte wird derzeit nicht verwendet, geben Sie daher keine Informationen auf dieser Registerkarte ein.
  11. Klicken Sie auf die Streams Registerkarte, auf der Sie einen Trailer für die Serie auswählen. Der folgende Screenshot hilft Ihnen bei der Auswahl dieses Trailers.
  12. Wähle Deine Anbieter. Höchstwahrscheinlich haben Sie nur einen Anbieter, der auf Ihr Brightcove Video Cloud-Konto verweist.
  13. Für die Stream-Video-ID , holen Sie sich die Video-ID für den Trailer von Studio und fügen Sie sie in das Textfeld ein.
  14. Seien Sie sicher, dass Stream-Typ ist eingestellt auf Anhänger.
  15. Klicken Sie auf die Rechte & Planung Registerkarte, wie hier gezeigt:

  16. Fügen Sie Werte für die folgenden Felder hinzu:

    Gebiet Erforderlich Anmerkungen
    Status Jawohl Wählen Sie einen der folgenden Typen:
    • Veröffentlicht
    • Unveröffentlicht
    Rechtetyp Jawohl Wählen Sie den passenden Wert für Ihre Serie aus den Optionen:
    • Werbung - AVOD
    • Kostenlos
    • Abonnement - SVOD
    Startzeit Jawohl Planen Sie einen Starttermin, an dem Ihre Serie verfügbar sein wird
    Endzeit Jawohl Planen Sie ein Enddatum, an dem Ihre Serie nicht mehr verfügbar ist
    Max. Streams Jawohl Legen Sie die maximale Anzahl von Streams fest, die die Serie von demselben Konto aus ansehen können
    Zu veröffentlichende Geräte Jawohl Fügen Sie die Geräte hinzu, auf denen Sie veröffentlichen möchten. Wähle aus Alle Geräte auswählen / abwählen um eine Liste der verfügbaren Geräte anzuzeigen
    Zulässige Standorte Jawohl

    Wählen Sie die Orte aus, an denen Sie Ihre Inhalte streamen möchten, wenn keine Einschränkungen eingegeben werden Welt.

    Wenn Sie nicht mindestens einen zulässigen Standort angeben, wird möglicherweise diese Meldung in Ihrer App angezeigt:
    Abgelehnte Standorte Nein Wählen Sie die Orte aus, an denen Sie Ihre Inhalte NICHT streamen möchten
  17. Klicken Sie unbedingt auf Neue Serie erstellen um die Serieninformationen zu speichern.

Sie haben nun Ihre Serie erstellt. Der nächste Schritt besteht darin, eine oder mehrere Staffeln zu erstellen, die zu dieser Serie gehören. Der nächste Abschnitt führt Sie durch diesen Prozess.

Jahreszeiten erstellen

Um eine Saison zu erstellen, gehen Sie folgendermaßen vor:

  1. Melden Sie sich bei Brightcove Beacon an und wählen Sie Programme.
  2. Wählen Sie in der linken Navigation Jahreszeiten.
  3. Drücke den Neue Saison hinzufügen Taste.
  4. Auf der Grundinformationen Registerkarte, füllen Sie Folgendes aus:
    Formularfeldname Beschreibung / Anforderungen
    Wählen Sie eine Serie Die Dropdown-Liste wird mit der von Ihnen erstellten Serie gefüllt. Wählen Sie die Serie aus, zu der diese Staffel gehört.
    Name Die Kennung der Serie, der kundenseitige Name wird später in der angegeben Textdaten Sektion
    Saisonnummer Dies muss eine ganze Zahl sein, die die Staffelreihenfolge in der Serie darstellt. Diese Nummer wird verwendet, wenn Episodenvideos von Video Cloud aufgenommen werden.
    Produktionsjahr Das Jahr, in dem die Saison produziert wurde; diese Informationen erscheinen in der Mehr Info Bildschirm.
    Produktionsland Das Land, in dem die Saison produziert wird; diese Informationen erscheinen in der Mehr Info Bildschirm.
  5. Klicken Sie auf die Textdaten Tab.
  6. Füllen Sie das aus Titel, Beschreibung und Zusammenfassung für die Saison. Diese Daten werden in der Brightcove Beacon-App angezeigt.
    Für die Saison Titel, Sie können einen Namen eingeben wie Staffel 1, aber wenn eine Serie Themen für verschiedene Jahreszeiten enthält, ist dies möglicherweise die bessere Wahl. Zum Beispiel eine Serie über die Kontinente, in der sich jede Staffel auf einen Kontinent konzentriert, gute Namen für die Jahreszeiten wären Europa, Afrika, Asien, etc.
  7. Klicken Sie auf die Nicht-textuelle Daten Tab.
  8. Liefern Sie für Ihre Saison ein Bild für Folgendes (entsprechend den unten stehenden Anforderungen):
    • Porträt-Poster (erforderlich)
    • Landschaftsposter (erforderlich)
    • Miniaturansicht
    • Breites Banner (erforderlich)

Einnahme von Episoden

Während das Erstellen der Serie und der zugehörigen Staffeln in Brightcove Beacon erfolgt, erfolgt die Erstellung der Episoden für die Serie in Video Cloud Studio. Sie weisen den einzelnen Episodenvideos in benutzerdefinierten Feldern Werte zu den Serien und Staffeln zu und nehmen diese Videos dann in Brightcove Beacon auf.

Denken Sie daran, dass der Inhalt der Brightcove Beacon-Episoden in Video Cloud liegt. Alle Änderungen, die Sie an Episoden in Brightcove Beacon vornehmen, werden überschrieben, wenn Sie Inhalte aus Video Cloud aufnehmen.

Voraussetzungen

Bevor Sie mit den folgenden Schritten fortfahren, stellen Sie sicher, dass Sie Folgendes ausgeführt haben:

  1. Melden Sie sich bei Video Cloud Studio an.

  2. Wählen Sie im Dropdown-Menü oben auf der Seite die Option HOCHLADEN.

  3. Laden Sie Ihre Episodenvideos hoch.

  4. Wählen Sie im Dropdown-Menü oben auf der Seite die Option MEDIEN.

  5. Aktualisieren Sie Ihre Episodenvideobilder wie in der Bilder und ihre Spezifikationen dokumentieren.

  6. Klicken Sie im Navigationskopf auf ADMINISTRATOR und dann Videofelder.

    Sie müssen ein Kontoadministrator sein, um auf die ADMINISTRATOR Speisekarte.
  7. Erstellen Sie die erforderlichen benutzerdefinierten Felder, die zum Aufnehmen von Videos in Brightcove Beacon erforderlich sind. Einzelheiten finden Sie im Erstellen benutzerdefinierter Video Cloud-Felder dokumentieren.

Schritte

Gehen Sie wie folgt vor, um Episodenvideos in Brightcove Beacon aufzunehmen:

  1. Kehren Sie zu Video Cloud Studio zurück.
  2. Wählen Sie im Dropdown-Menü oben auf der Seite die Option MEDIEN.

  3. Klicken Sie auf den Titel eines der Episodenvideos, um die Eigenschaftenseite des Videos aufzurufen.

  4. Scrollen Sie nach unten, bis Sie auf der rechten Seite die BENUTZERDEFINIERTE FELDER Sektion.
  5. Drücke den Bearbeiten Taste.
  6. Suchen Sie die drei benutzerdefinierten Felder, die Episoden zugeordnet sind.
  7. Geben Sie für das Episodenvideo die Episodennummer, Staffelnummer und den Seriennamen in die entsprechenden benutzerdefinierten Felder ein.
  8. Klicken Sie unbedingt auf Speichern -Taste, nachdem Sie die entsprechenden Werte für das Video eingegeben haben.
  9. Wiederholen Sie die Eingabe von Werten in die benutzerdefinierten Felder für alle Ihre Episodenvideos.
  10. Melden Sie sich bei Brightcove Beacon an.
  11. Klicken Sie auf die Werkzeuge Möglichkeit (Schlüssel Symbol) im oberen Menü.

  12. Seien Sie sicher, dass Einnahme Option wird im linken Menü ausgewählt.

  13. Drücke den Brightcove-Videos aktualisieren um die neu konfigurierten Episodenvideos aufzunehmen.

  14. Um die Videos sofort in Ihren Apps verfügbar zu machen, leeren Sie den Cache von Brightcove Beacon, indem Sie auf das Zwischenspeicher Option in der linken Navigation, und klicken Sie dann auf die Cache-Bereinigung Taste.

    Weitere Informationen zur Verwendung der Brightcove Beacon Tools-Optionen finden Sie im Verwenden der Tools dokumentieren.

Damit sind die Informationen zum Erstellen von Serien und den dazugehörigen Staffeln und Episoden abgeschlossen.
