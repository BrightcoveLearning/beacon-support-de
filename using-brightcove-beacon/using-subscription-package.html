    Verwendung eines Abonnementpakets

    In diesem Thema erfahren Sie, wie Sie ein Video-Abonnementpaket verwenden, das mit der Registerkarte „ Commerce “ von Brightcove Beacon erstellt wurde. Das Abonnementpaket implementiert die SVOD-Monetarisierung (Abonnementvideo on Demand). Einzelheiten zum Erstellen des Abonnements finden Sie im Dokument Erstellen eines Abonnementpakets .

    Übersicht

    Ein Abonnementpaket definiert, wie ein bestimmter Satz von Videos von den Kunden Ihrer Apps gekauft werden kann. Das Abonnementpaket muss in Brightcove Beacon erstellt werden und wird im Dokument Creating a Subscription Package beschrieben. Entscheidend für die Verwendung des Abonnementpakets sind auch die benutzerdefinierten Felder von Video Cloud, die für Videos in einem Video Cloud-Konto erstellt wurden, wie im Dokument Creating Video Cloud Custom Fields beschrieben. Insbesondere werden die Felder Rights Management verwendet.

    Verstehen der benutzerdefinierten Felder für die Rechte

    Die benutzerdefinierten Felder für die Rechteverwaltung bestehen aus:

    • beacon.Rechte.<zähler > .type
    • beacon.Rechte.<Zähler > .startDate
    • beacon.Rechte.<Zähler > .endDate
    • beacon.Rechte.<zähler > .devices
    • beacon.Rechte.<zähler > .locationsPermit
    • beacon.Rechte.<zähler > .locationsDeny
    • beacon.Rechte.<zähler > .packageName
    • beacon.Rechte.<Zähler > .adConfiguration

    Betrachten Sie zuerst den < > Zählerteil jedes Namens. Dies ist enthalten, da Sie möglicherweise mehrere Gruppen von Rechteverwaltungsfeldern erstellen möchten, und der < Zähler > wird verwendet, damit Sie die Array-Typsyntax verwenden können, um zugehörige Felder zu gruppieren.

    Angenommen, Sie wollten Abonnementpakete rund um die Feiertage erstellen und Pakete für:

    • Tag der Erde
    • Arbor-Tag
    • Neue Jahre

    Sie müssen in Video Cloud gehen und drei Sätze von Rechteverwaltungsfeldern erstellen, wobei Sie Leistungsindikatorwerte von 0-2 (oder 1-3, wenn Sie dies bevorzugen) verwenden, die wie hier gezeigt angezeigt werden:

    Zusammenfassend müssen Sie für jedes Paket, das Sie verwenden möchten, einen „Zähler“ -Satz von Rechteverwaltungsfeldern erstellen.

    Paket- und benutzerdefiniertes Feld ähnliche Rechte

    Einige benutzerdefinierte Felder, denen Sie Daten zuweisen und bei der Aufnahme auf ein Video angewendet haben, können beim Erstellen Ihres Pakets in Brightcove Beacon genauso aussehen wie Formularfelder. Zum Beispiel das benutzerdefinierte Feld beacon.rights.<counter > .locationsDeny und das Formularfeld Verweigerte Standorte des Pakets scheinen in Konflikt zu geraten. Denken Sie daran, dass bei der Betrachtung dieser „Konflikte“ zwei verschiedene Fragen gestellt werden:

    • Erlauben die Rechte an dem Video das Abspielen?
    • Darf ein Benutzer das Paket abonnieren?

    Bedenken Sie diese Szenarien:

    • Ein Enddatum im Beacon.Rights.< >counter.endDate benutzerdefiniertes Feld liegt später als das Enddatum des Pakets selbst. Dies bedeutet, dass die Rechte des Videos es ermöglichen würden, bis zum Enddatum des benutzerdefinierten Feldes angezeigt zu werden, aber der Benutzer konnte das Paket nicht abonnieren, das beispielsweise die Kosten für die Anzeige des Videos gesenkt haben könnte.
    • Es wird ein Paket erstellt, das zugelassene Standorte von USA, Kanada und Mexiko hat, und dieses Paket wird dem Video im benutzerdefinierten Feld beacon.rights zugewiesen.<zähler > .packageName. Auch in den benutzerdefinierten Feldern des Videos wird Großbritannien in die Beacon.rights eingefügt.< >counter.locationsPermit benutzerdefiniertes Feld. Das Paket ist dem Video zugewiesen, aber das Paket kann nicht in Großbritannien gekauft werden. Das Video könnte in Großbritannien abgespielt werden, aber es könnte nicht Teil des Pakets für einen britischen Kunden sein.
    • Es wird ein Paket erstellt, das Standorte von USA, Kanada und Mexiko zugelassen hat. In den benutzerdefinierten Feldern werden die USA und Kanada in den Beacon.rights des Videos hinzugefügt.< >counter.locationsDeny benutzerdefiniertes Feld. Hier könnten die Kunden in den USA, Kanada und Mexiko das Paket abonnieren, aber das spezifische Video, in dem den USA und Kanada den Standorten verweigert werden, bedeutet, dass das Video von Kunden an diesen Orten nicht angesehen werden konnte, obwohl das Video im Paket enthalten ist.

    Zuweisen eines einzelnen Videos zu einem Paket

    Um einem Paket ein einzelnes Video zuzuweisen, müssen Sie Video Cloud Studio verwenden. Führen Sie die folgenden Schritte durch:

    1. Navigieren Sie in Studio zum Medienmodul .
    2. Klicken Sie auf den Titel des Videos, den Sie dem Paket hinzufügen, um die Eigenschaften dieses Videos zu sehen.
    3. Scrollen Sie nach unten, bis Sie auf der rechten Seite den Abschnitt Benutzerdefinierte Felder sehen, und klicken Sie auf die Schaltfläche Bearbeiten .
    4. Geben Sie den Namen des in Brightcove Beacon erstellten Abonnementpakets in das beacon.rights.<counter>.packageName Feld ein.
    5. Da Sie ein Paket verwenden, MÜSSEN Sie das beacon.rights.<counter>.type auf SVODsetzen.
    6. Scrollen Sie zum Ende der Liste der benutzerdefinierten Felder und klicken Sie auf die Schaltfläche Speichern .

    Zuweisen mehrerer Videos zu einem Paket

    Um einem Paket mehrere Videos zuzuweisen, müssen Sie Video Cloud Studio verwenden. Der folgende Screenshot ist bei der Durchführung der Schritte hilfreich:

    1. Navigieren Sie in Studio zum Medienmodul und aktivieren Sie das Kontrollkästchen vor jedem Video, das Sie dem Paket hinzufügen möchten.
    2. Klicken Sie auf die Schaltfläche „Quick Edit
    3. Der Abschnitt „Schnellbearbeitung “ wird auf der rechten Seite des Bildschirms angezeigt, und dort können Sie den Namen des in Brightcove Beacon erstellten Abonnementpakets in das beacon.rights.<counter>.packageName Feld eingeben.
    4. Da Sie ein Paket verwenden, MÜSSEN Sie das beacon.rights.<counter>.type auf SVODsetzen.
    5. Klicken Sie unten im Abschnitt „Quick Edit “ auf die Schaltfläche „ Speichern “.

    Aufnahme der Videos in Brightcove Beacon

    Sie haben jetzt ein oder mehrere Videos mit Ihrem Abonnementpaket verknüpft. Sie müssen diese Assoziationen jetzt in Brightcove Beacon aufnehmen. Gehen Sie dazu folgendermaßen vor:

    1. Melden Sie sich bei Brightcove Beacon an.
    2. Klicken Sie auf das Schraubenschlüssel-Symbol (Tools).
    3. Klicken Sie auf der linken Navigationsleiste auf Verschlucken.
    4. Klicken Sie auf die Schaltfläche „ Brightcove Videos aktualisieren “.
    5. Wenn Fehler gemeldet werden, überprüfen Sie diese, um sicherzustellen, dass keine mit Ihrem Abonnementpaket in Verbindung stehen.
    6. Wenn Sie möchten, dass die neuen Verbände sofort in Brightcove Beacon aktiv sind, müssen Sie den aktuellen Cache löschen. Klicken Sie dazu in der linken Navigationsleiste auf Cache und dann auf Cache bereinigen.

    Weitere Informationen zu den in diesem Abschnitt durchgeführten Aktionen finden Sie im Dokument Verwenden der Tools .
